Scan Tool Data Definitions
Input Shaft Revolution (rpm)
Input shaft revolutions calculated by reference pulses coming from input shaft speed sensor in A/T.
Output Shaft Revolution (rpm)
Output shaft revolutions calculated by reference pulses coming from output shaft speed sensor in A/T.
ATF Temperature (°C, °F)
ATF temperature calculated by signal from ATF temperature sensor installed in valve body.
Battery Voltage (V)
This parameter indicates battery voltage inputted through ignition switch to TCM.
Selector Position Sensor No.1 (V)
This parameter indicates output voltage of transmission range sensor (main).
Selector Position Sensor No.2 (V)
This parameter indicates output voltage of transmission range sensor (sub).
Line Pressure Solenoid Control Value (mA)
This parameter indicates control current value that TCM outputs to line pressure control solenoid valve.
A Solenoid (Com) (Pressure Control Solenoid A Command, mA)
This parameter indicates control current value that TCM outputs to pressure control solenoid valve “A”.
B Solenoid (Com) (Pressure Control Solenoid B Command, mA)
This parameter indicates control current value that TCM outputs to pressure control solenoid valve “B”.
C Solenoid (Com) (Pressure Control Solenoid C Command, mA)
This parameter indicates control current value that TCM outputs to pressure control solenoid valve “C”.
D Solenoid (Com) (Pressure Control Solenoid D Command, mA)
This parameter indicates control current value that TCM outputs to pressure control solenoid valve “D”.
TCC Solenoid (Com) (TCC Solenoid Command, mA)
This parameter indicates control current value that TCM outputs to TCC solenoid valve.
Electro Magnetic Oil Pump Solenoid Duty (%)
This parameter indicates solenoid duty ratio that TCM outputs to electro magnetic oil pump solenoid valve.
Line Pressure Solenoid Monitor Value (mA)
This parameter indicates current value that flows to line pressure control solenoid valve.
A Solenoid (Mon) (Pressure Control Solenoid A Monitor, mA)
This parameter indicates current value that flows to pressure control solenoid valve “A”.
B Solenoid (Mon) (Pressure Control Solenoid B Monitor, mA)
This parameter indicates current value that flows to pressure control solenoid valve “B”.
C Solenoid (Mon) (Pressure Control Solenoid C Monitor, mA)
This parameter indicates current value that flows to pressure control solenoid valve “C”.
D Solenoid (Mon) (Pressure Control Solenoid D Monitor, mA)
This parameter indicates current value that flows to pressure control solenoid valve “D”.
TCC Solenoid (Mon) (TCC Solenoid Monitor, mA)
This parameter indicates current value that flows to TCC solenoid valve.
Engine Speed (rpm)
Engine speed calculated by signal through CAN communication line fed from ECM.
Accel Actual Position (Accelerator Actual Position, %)
Accelerator pedal position calculated by signal through CAN communication line fed from ECM.
Accel Effective Position (Accelerator Effective Position, %)
This parameter indicates accelerator pedal position calculated by accelerator pedal position sensor input and cruise control.
ECT (Engine Coolant Temperature, °C, °F)
This parameter indicates engine coolant temperature sent from ECM through CAN.
Transaxle Range (P / R / N / D / Manual)
This parameter indicates transaxle range detected by signal transmitted from transmission range sensor.
Actual Gear Position (1st / 2nd / 3rd / 4th / 5th / 6th / R / N)
This parameter indicates the gear position of A/T.
Torque Converter Clutch (Release / Slip / Connect)
This parameter indicates the status of lock-up clutch output from TCM.
Neutral Control Mode (Neutral Control Not Active / Entering Neutral Control / Neutral Control Active / Exiting Neutral Control)
This parameter indicates the status of neutral control.
Slip RPM (rpm)
This parameter indicates slipping rotation in the torque converter (speed difference between input shaft rotation and engine rotation)
Vehicle Speed (km/h, MPH)
This parameter indicates vehicle speed calculated by output shaft speed sensor signal.
Paddle Up SW (Paddle Up Switch, ON / OFF)
This parameter indicates the input status of paddle-up switch “+” to TCM.
Paddle Down SW (Paddle Down Switch, ON / OFF)
This parameter indicates the input status of paddle-down switch “–” to TCM.
Paddle Up SW (Mon) (Paddle Up Switch Monitor, ON / OFF)
This parameter indicates the status of paddle-up switch “+”.
Paddle Down SW (Mon) (Paddle Down Switch Monitor, ON / OFF)
This parameter indicates the status of paddle-down switch “–”.
Shift Solenoid A (Com) (Shift Solenoid A Command, ON / OFF)
This parameter indicates control signal output from TCM to shift solenoid valve “A”.
Shift Solenoid B (Com) (Shift Solenoid B Command, ON / OFF)
This parameter indicates control signal output from TCM to shift solenoid valve “B”.
Shift Solenoid A (Mon) (Shift Solenoid A Monitor, ON / OFF)
This parameter indicates the status of shift solenoid valve “A”.
Shift Solenoid B (Mon) (Shift Solenoid B Monitor, ON / OFF)
This parameter indicates the status of shift solenoid valve “B”.
Neutral Control Inhibit (Permit / Inhibit)
This parameter indicates the permission status of neutral control.
Brake Switch (ON / OFF)
Brake light switch position detected by signal transmitted from ECM through CAN communication line.
ON: Brake pedal depressed
OFF: Brake pedal released
Air Conditioner Signal (ON / OFF)
ON: Command for A/C operation is being output from HVAC to ECM.
OFF: Command for A/C operation is not being output.
Transaxle Warning Light (OFF / ON)
This parameter indicates the status of transmission warning light.
MIL Request (OFF / ON)
This parameter indicates the status of MIL.
Start Lock Output (OFF / ON)
This parameter indicates the status of engine start permission signal to starting motor control relay.
Reverse Lamp Output (OFF / ON)
This parameter indicates the status of back up light relay drive signal.
