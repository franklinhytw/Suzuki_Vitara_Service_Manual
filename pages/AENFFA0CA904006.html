Fail-Safe Table
When a DTC is detected, the ENG A-STOP control module switches to the fail-safe mode and stays in the mode as long as faulty condition persists but the fail-safe mode is canceled when the ENG A-STOP control module detects normal condition.
NOTE:
The fail-safe table below lists the fail-safe operations that are performed when DTC is detected during the engine auto stop control.
If a DTC is detected while the engine is running, the ENG A-STOP control module suspends the ENG A-STOP system.
