Input Shaft Speed Sensor Removal and Installation
Removal
1)Remove valve body assembly.  
2)Disconnect input shaft speed sensor connector (1).
3)Remove input shaft speed sensor (2).
Installation
Reference: Input Shaft Speed Sensor Inspection
Reverse removal procedure noting the following point.
•Tighten input shaft speed sensor bolt (1) to specified torque.
Tightening torque
Input shaft speed sensor bolt (a): 5.4 N·m (0.55 kgf-m, 4.0 lbf-ft)
