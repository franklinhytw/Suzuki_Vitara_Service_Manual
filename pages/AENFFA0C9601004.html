9F
Power Door Lock Operation Description
Locking / unlocking of doors is conducted by operation of the items shown in the following table.
NOTE:
Power door lock operation can also be selected by setting change function of SUZUKI scan tool.  
1 action type
Item Activated door lock(s)
Remote controller or transmitter All doors
Door request switch
Power door lock switch (Driver side)
Key cylinder
Door lock lever Operated door
2 action type
Item Activated door lock(s)
Remote controller or transmitter Push the LOCK button once and all doors are locked
Push the UNLOCK button once and driver side door is unlocked
Push the UNLOCK button twice and all doors are unlocked
Door request switch Operation Lock Push the request switch once and all doors are locked
Unlock Push the request switch once and operated door is unlocked
Push the request switch twice and all doors are unlocked
Power door lock switch (Driver side) All doors
Key cylinder Turn the key cylinder to LOCK position once and all doors are locked
Turn the key cylinder to UNLOCK position once and driver side door is unlocked
Turn the key cylinder to UNLOCK position twice and all doors are unlocked
Door lock lever Operated door
