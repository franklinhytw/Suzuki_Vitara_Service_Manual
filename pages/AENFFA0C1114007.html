NOTE:
•If communication between SUZUKI scan tool and ECM is not possible, perform the following checks:
—Check communication condition between PC and VCI using SUZUKI scan tool function.
—Check DLC power circuit and ground circuit.
—Check DLC cable. If necessary, check DLC cable by substituting a known-good DLC cable.
—Check CAN communication system. (“Control Module Communication Bus Off” is detected in any control module.)  
—Check ECM power circuit and ground circuit.  
•For type distinction of OBD system, refer to “Discrimination for OBD System” under OBD System Description:M16A.

•When checking data with engine running at idle or racing, place gear shift lever in neutral position and apply parking brake lever fully. Also, if nothing or “no load” is indicated, turn off A/C, all electric loads, P/S and all the other switches.

Scan Tool Data Definitions
Fuel System (Fuel system status)
A/F ratio feedback loop status is displayed as follows.
OL: Open loop control before satisfaction of closed loop control conditions (ECT > 10 °C, IAT, TP sensor/system = OK).
CL: Closed loop control using oxygen sensor(s) as feedback sensor(s).
OL-DRIVE: Open loop control due to driving requirement (power-up enrichment etc.).
OL-FAULT: Open loop control system fault.
CL-FAULT: Closed loop control system fault.
Short Term Fuel Trim (%)
Short term fuel trim value represents short term corrections to air/fuel mixture. 0 (zero) indicates no correction, a value greater than 0 indicates enrichment correction, and a value less than 0 indicates enleanment correction.
Long Term Fuel Trim (%)
Long term fuel trim value represents long term corrections to air/fuel mixture. 0 (zero) indicates no correction, a value greater than 0 indicates enrichment correction, and a value less than 0 indicates enleanment correction.
Total Fuel Trim (%)
Total fuel trim is calculated from short term fuel trim and long term fuel trim. This value indicates how much correction is necessary to keep the air/fuel mixture stoichiometrical.
Inj Pulse Width (Fuel injection pulse width, msec)
This parameter indicates the injector drive pulse width (time) during which the injector valve stays open as commanded by ECM. (For multipoint fuel injection, the injector drive time means that of the No.1 cylinder.)
Ignition Advance (Ignition timing advance for No.1 cylinder, ° BTDC)
This parameter indicates the ignition timing of No.1 cylinder commanded by ECM. The actual ignition timing should be checked using a timing light.
Calculated Load (Calculated load value, %)
This parameter indicates calculated engine load in percent. The load is calculated based on input signals to ECM from various engine sensors.
Engine Speed (rpm)
The engine speed is calculated from reference pulses from the camshaft position sensor.
Desired Idle (Desired idle speed, rpm)
The desired idle speed is an ECM internal parameter which indicates the idle speed requested by ECM. If the engine is not running, this number is not valid.
Vehicle Speed (km/h, MPH)
The vehicle speed is calculated based on pulse signals from wheel speed sensors.
MAP (Manifold absolute pressure, kPa, mmHg, in.Hg, psi, inH2O)
This parameter indicates the calculated manifold pressure based on MAP sensor signal.
ECT (Engine coolant temperature, °C, °F)
It is detected by engine coolant temperature sensor.
Intake Air Temperature. (°C, °F)
It is detected by intake air temperature sensor.
Barometric Pres (Barometric pressure, kPa, mmHg, in.Hg, psi, inH2O)
This parameter represents barometric air pressure measurement and is used for altitude correction of the fuel injection rate.
Evap Canist Prg Duty (Evap canister purge flow duty, %)
This parameter indicates valve ON (valve open) time rate within a certain set cycle of EVAP canister purge valve which controls the amount of EVAP purge.
VVT Gap (Variable valve timing difference between target and actual position, °)
This parameter indicates difference between target camshaft position (intake side) and actual camshaft position (intake side).
IAC Throttle Opening (Idle air throttle valve opening, %)
This parameter indicates throttle valve opening of idle air control in % (100% indicates the maximum idle air flow).
TP Sensor 1 Volt (TP sensor (main) output voltage, V)
TP Sensor (Main) data provides throttle valve opening information in the form of voltage.
TP Sensor 2 Volt (TP sensor (sub) output voltage, V)
TP Sensor (Sub) data provides throttle valve opening information in the form of voltage.
APP Sensor 1 Volt (APP sensor (main) output voltage, V)
APP sensor (main) data provides accelerator pedal position information in the form of voltage.
APP Sensor 2 Volt (APP sensor (sub) output voltage, V)
APP sensor (sub) data provides accelerator pedal position information in the form of voltage.
Accel Position (Absolute accelerator pedal position, %)
This parameter indicates calculated accelerator pedal position in percent based on input from accelerator pedal position sensor.
Throttle Position (Absolute throttle position, %)
This parameter indicates calculated throttle opening in percent based on input from throttle position sensor.
Target Throttle Posi (Target throttle valve position, %)
Target throttle valve position is an ECM internal parameter which indicates the throttle valve position requested by ECM.
O2S B1 S1 (HO2S-1, V)
This parameter indicates output voltage of HO2S-1 installed on exhaust manifold (pre-TWC).
O2S B1 S2 (HO2S-2, V)
This parameter indicates output voltage of HO2S-2 installed on exhaust No.1 pipe (post-TWC). It is used to detect catalyst deterioration.
Battery Voltage (V)
This parameter indicates battery voltage inputted from main relay to ECM.
Fuel Tank Level (%)
This parameter indicates approximate fuel level in the fuel tank. The fuel level data is provided in percentage from 0 to 100%.
A/C Pressure (A/C refrigerant absolute pressure, kPa)
This parameter indicates A/C refrigerant absolute pressure detected by A/C refrigerant pressure sensor.
Generator Reg Volt (Com) (Generator Regulated Voltage (Com) (V))
This parameter indicates commanded regulated-voltage to generator.
Radiator Fan (Radiator cooling fan control relay, Low/High/Off)
Low: ON command is being output to the radiator cooling fan relay No.1.
High: ON command is being output to radiator cooling fan relays No.2 and No.3.
Off: No command is being output.
A/C Comp Relay (A/C compressor relay, ON/OFF)
This parameter indicates the state of the A/C compressor relay.
Fuel Pump (ON/OFF)
ON is displayed when ECM activates the fuel pump via the fuel pump relay switch.
Intake Tuning Sol (Com) (IMT valve actuator control command, Open / Close)
Open: Open command is being output to IMT valve actuator.
Close: Close command is being output to IMT valve actuator.
Starter SW (Starter switch, ON/OFF)
This parameter indicates state of starting motor control relay output.
ON: Starting motor control relay is on.
OFF: Starting motor control relay is off.
A/C Switch (ON/OFF)
ON: Command for A/C operation is being output from HVAC to ECM.
OFF: Command for A/C operation is not being output.
Blower Fan (ON/OFF)
This parameter indicates the state of the blower speed selector.
Brake Switch (ON/OFF)
This parameter indicates the state of the brake switch.
Electric Load (ON/OFF)
ON: ON signal of headlight, clearance lights or rear end door window defogger is input.
OFF: Above electric loads all turned off.
PNP Signal (P/N or D range)
Whether the transmission range sensor is in P/N range or other ranges is displayed.
If it is in P or N range, “P/N” is displayed and if it is in other than P or N ranges, “D” is displayed.
Clutch SW (Declutch / Clutched)
Declutch: Clutch pedal is fully depressed.
Clutched: Clutch pedal is released.
Cruise RES/ACC Switch (“RES / +” switch, OFF/ON)
ON: “RES / +” switch is pushed.
OFF: “RES / +” switch is released.
Cruise SET/COAST Switch (“SET / –” switch, OFF/ON)
ON: “SET / –” switch is pushed.
OFF: “SET / –” switch is released.
Cruise CANCEL Switch (“CANCEL” switch, OFF/ON)
ON: “CANCEL” switch is pushed.
OFF: “CANCEL” switch is released.
Cruise ON/OFF Switch (Released / Pushed)
Pushed: “Cruise” switch is pushed.
Released: “Cruise” switch is released.
Distance Setting Switch (OFF/ON)
ON: Distance switch is pushed.
OFF: Distance switch is released.
Fuel Cut (ON/OFF)
ON: Fuel injection is cut (output signal to injector is stopped).
OFF: Fuel injection is not cut.
Closed Throttle Pos (Closed throttle position, ON/OFF)
This parameter is ON when throttle valve is fully closed and OFF when the valve is not fully closed.
O2S B1 S1 ACT (HO2S-1, Active / Inactive)
This parameter indicates the activation condition of HO2S-1.
Active: The sensor is activated.
Inactive: The sensor is being warmed up or not activated.
O2S B1 S2 ACT (HO2S-2, Active / Inactive)
This parameter indicates the activation condition of HO2S-2.
ACTIVE: The sensor is activated.
INACTIVE: The sensor is being warmed up or not activated.
Intake Valve Actuator (IMT valve actuator, Open / Closed)
This parameter indicates the state of IMT valve actuator.
ENG A-STOP (Mon, Engine running / Stopping)
This parameter indicates the state of the engine auto-stop.
ENG A-STOP (Com, Request / Not request)
This parameter indicates the state of the engine auto-stop request
