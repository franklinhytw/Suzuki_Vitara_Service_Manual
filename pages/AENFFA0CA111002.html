10A
Description of Cruise Control System Components and Functions
Component Function
ECM and electric throttle body (M16A model / K14C model) or fuel injector (D16AA model) ECM performs centralized control over all functions including setting a constant speed, resuming it, setting coast, and canceling cruise control minimum limit speed.
ECM controls electric throttle valve opening (M16A model / K14C model) or fuel injector (D16AA model) to keep actual vehicle speed at set (target) speed.
CRUISE (main) switch This switch has a momentary contact type button to turn cruise control function ON and OFF.
SET/– switch When this switch is pressed (ON) and then released (OFF) while vehicle is running at a speed of 40 km/h (25 mile/h) or higher, vehicle speed at that OFF moment is stored in memory and it is maintained (constant cruising).
When this switch is continuously pressed (ON) during constant cruising, vehicle speed will continually decrease as long as switch is ON. When it is released (OFF), vehicle speed at that moment is stored in memory and vehicle starts constant cruising.
Pressing this switch (ON) during constant cruising decreases vehicle speed according to number of times the switch is pressed.
RES/+ switch When this switch is pressed (ON) during constant cruising, vehicle speed will continually increase as long as switch is ON. When it is released (OFF), vehicle speed at that moment is stored in memory and vehicle starts constant cruising. If vehicle speed is higher than 40 km/h (25 mile/h) after cruise control is canceled, pressing this switch ON momentarily will resume vehicle running at the speed.
Pressing this switch (ON) during constant cruising increases vehicle speed according to number of times the switch is pressed.
CANCEL switch When this switch is pressed (ON), cruise control (throttle valve control (M16A model / K14C model) or fuel injector control (D16AA model)) is canceled.
Wheel speed sensor (vehicle speed signal) ECM receives speed sensor signal from ESP® control module through CAN communication and calculates vehicle speed using that signal.
Brake light switch Brake light switch has 2 contact points. One contact point closes when brake pedal is depressed to light brake light and provide a voltage signal to the ECM.
The other contact point (brake pedal position switch) opens when brake pedal is depressed to shut off power to cruise control of ECM, thereby canceling cruise control (throttle valve control (M16A model / K14C model) or fuel injector control (D16AA model)).
This switch is installed to cancel cruise control.
Clutch pedal position switch (M/T model) When clutch pedal is depressed, clutch pedal position switch closes and provides a ground signal to ECM.
ECM cancels cruise control (throttle valve control (M16A model / K14C model) or fuel injector control (D16AA model)) when this signal is inputted.
TCM (A/T model / twin clutch system model) TCM transmits transmission shift position signal to ECM through CAN communication.
“CRUISE” indicator In the state with ignition “ON” and cruise control system in OFF, pressing CRUISE (main) switch once and releasing it will activate the cruise control system and ECM will cause indicator to light up.
“SET” indicator Lights up when cruise control (throttle valve control (M16A model / K14C model) or fuel injector control (D16AA model)) is functioning.
