10B
BCM Self-Diagnosis Function
Description
BCM monitors conditions of system components and its circuit when ignition is “ON”. When an abnormal condition in the system occurs, BCM performs controls as follows.
Self-Diagnosis Function
When an abnormal condition in the system occurs, DTC is stored in the memory of EEPROM in BCM.
Fail-Safe Control
When DTC is detected, BCM enters fail-safe mode as long as malfunction continues, but that mode is canceled when BCM recovers normal condition.
