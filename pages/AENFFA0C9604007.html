Scan Tool Active Test Table
SUZUKI scan tool has output test function which can force operation of the following actuators and relays of the system controlled by keyless entry control module. When a malfunction is found in the system controlled by keyless entry control module, execute output test which facilitates judgment whether malfunction is on input side or output side of keyless entry control module. For details of SUZUKI scan tool, refer to its Operator's Manual.
NOTE:
Select “Keyless Start” menu on SUZUKI scan tool.
