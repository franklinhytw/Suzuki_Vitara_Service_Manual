10A
Adaptive Cruise Control System Self-Diagnosis Function
Description
RBS control module and ECM monitor conditions of adaptive cruise control system components and circuits. When an abnormal condition in the system occurs, RBS control module and ECM operate as follows.
Self-Diagnosis Function
RBS control module and ECM have self-diagnosis function that stores DTC for abnormalities detected in adaptive cruise control system. Stored DTCs remain in memory of RBS control module and ECM unless they are cleared.
NOTE:
RBS control module does not store history DTCs of C1610 and C1640.
Warning Function
RBS control module turns on the adaptive cruise control warning light to warn the driver that a malfunction is detected.
Fail-Safe Control
When any abnormality is found, RBS control module disables adaptive cruise control.
