APP Sensor On-Vehicle Inspection
NOTE:
After replacing accelerator pedal assembly (incorporating APP sensor), calibrate throttle valve position.  
1)Check that accelerator pedal assembly (incorporating APP sensor) has been mounted to vehicle body properly (no pinched floor carpet, etc.).
If mounting is improper, reinstall accelerator pedal assembly.  
2)Connect scan tool to DLC with ignition “OFF”.
3)Set ignition “ON” and select “Data List” mode.
4)Check that voltage varies depending on accelerator pedal position as shown in the following graph.
If faulty condition is found, check the following points.
•Wire harness and connections
•APP sensor  
[a]: APP sensor (main) voltage
[b]: APP sensor (sub) voltage
[c]: Voltage
[d]: Idle position of accelerator pedal
[e]: Fully depressed position of accelerator pedal
