Back Up Light Removal and Installation
Removal
1)Remove rear end door trim.  
2)Disconnect back up light connector (1).
3)Turn socket counterclockwise to remove bulb, if necessary.
4)Remove back up light nut (2).
5)Pull back up light (1) in arrow direction shown in figure to disengage stud pins (2) and remove back up light.
3. Grommet
Installation
Reverse removal procedure noting the following point.
•Tighten back up light nut to specified torque.
Tightening torque
Back up light nut (a): 6.0 N·m (0.61 kgf-m, 4.5 lbf-ft)
