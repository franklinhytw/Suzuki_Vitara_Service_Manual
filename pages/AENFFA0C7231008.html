OBD System Description
HVAC control module detects malfunctions, which may occur in the following areas:
•Inside air temperature sensor
•Sunload sensor
•CAN communication line
•LIN communication line
•Temperature control actuator
•Air flow control actuator
•Air intake control actuator
•Auto A/C panel
•HVAC control module
When HVAC control module detects malfunction, the “AUTO” display (1) flashes to warn and the diagnostic trouble code (DTC) is stored in the memory of the module. When diagnosing trouble, the DTC can be checked with DTC Check:Automatic Type.
