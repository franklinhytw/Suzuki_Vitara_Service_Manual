1C
Fuel Injection System Part Parameter Initialization
After replacing one or more fuel injector(s), MAF sensor with IAT sensor-1, fuel pressure sensor and/or common rail with new one, perform the following procedure.
1)When ignition is “OFF”, connect SUZUKI scan tool to DLC.
2)Set ignition “ON”.
3)Execute “Fuel injection system part parameter initialization” in “Utility” menu following instructions displayed on SUZUKI scan tool.
4)When ignition is “OFF”, disconnect SUZUKI scan tool from DLC.
