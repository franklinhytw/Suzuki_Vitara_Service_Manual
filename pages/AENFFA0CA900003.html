Precautions for Replacing ENG A-STOP Control Module
If communication between SUZUKI scan tool and ENG A-STOP control module is not possible, first check related items according to the following points for recovery.
•SUZUKI scan tool and communication cable
•DLC power circuit and ground circuit
•CAN communication system malfunction (“Control Module Communication Bus Off” is detected in any control module)
If communication is still impossible, replace starting motor as well.
