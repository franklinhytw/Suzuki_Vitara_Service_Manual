Shift Lock Solenoid Inspection
NOTICE:
Short circuit or an improper connection will damage the shift lock solenoid.
Check that the power supply voltage and test connections are both correct.
1)Remove floor console front cover.  
2)Disconnect select lever connector (1).
3)Check that shift lock solenoid rod moves smoothly and then returns when positive terminal of battery is connected to terminal “5” and negative terminal of battery is connected to terminal “10”.
Replace select lever assembly, if solenoid rod does not move smoothly.
Select lever circuit
[A]: Keyless push start model 4. To keyless start control module
2. P position switch 5. To BCM
3. Shift lock solenoid
