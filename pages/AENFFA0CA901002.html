Description of ENG A-STOP System Components
The ENG A-STOP system consists of the following components.
Component Description
ENG A-STOP control module This module incorporating DC/DC converter is located inside the instrument panel on the right side. The module communicates with the ECM, ESP® control module, HVAC control module (auto A/C model), combination meter, etc. via the CAN communication line. The module performs the ENG A-STOP controls according to signals from the relevant sensors and switches.
CPP No.1 switch (M/T model)
CPP No.2 switch (M/T model)		 These switches are installed on the clutch pedal bracket. The switches provide the ENG A-STOP control module with information on the state of the clutch pedal.
Neutral position switch (M/T model) This switch is installed on the transaxle case. The ENG A-STOP control module determines whether the gear is in neutral based on the signal from this switch.
Electric load current sensor & battery temperature sensor This sensor is fitted on the battery negative (–) terminal. The sensor provides information on the temperature around the battery and battery current to the ENG A-STOP control module.
Hood latch switch This switch is installed on the hood latch. The ENG A-STOP control module identifies the open / closed state of the hood according to the signal from this switch.
DC/DC converter (A/T model) This DC/DC converter for A/T is located inside the instrument panel on the driver’s side. The converter is controlled by the ENG A-STOP control module and functions to prevent voltage drop of power supply for the TCM when the starting motor starts up for restarting the engine.
ENG A-STOP OFF switch The ENG A-STOP OFF switch is installed in the switch bezel at the instrument panel. Setting this switch to the operating position enables the driver to cancel the engine auto stop control.
This switch has a momentary contact type button to turn engine auto stop control function ON and OFF.
ENG A-STOP indicator light The ENG A-STOP indicator light is located in the combination meter. This indicator light lights up during the engine auto stop control. This light blinks 2 times when the engine auto start control is activated at specified condition.
ENG A-STOP warning light This warning light is located in the combination meter. The light blinks when there is a problem in the system. The light lights up when the ENG A-STOP OFF switch is pushed and the ENG A-STOP control module stops the engine auto stop control.
Buzzer This buzzer is located in the combination meter. The buzzer sounds when any of the conditions for inhibiting the engine auto start control is met.
ST2 relay The ST2 relay is in the relay box. When the ENG A-STOP control module restarts the engine, the module turns on the relay so that electric power is supplied to the starting motor.
Brake light switch (A/T model / twin clutch system model) This switch is installed on the brake pedal assembly. The switch provide the ENG A-STOP control module with information on the state of the brake pedal.
