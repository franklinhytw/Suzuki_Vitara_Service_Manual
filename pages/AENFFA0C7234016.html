B1504-H: Sunload Sensor and/or Its Circuit Malfunction
Signal voltage of position sensor in sunload sensor is higher than specified value (4.9 V) for specified time continuously.
•Sunload sensor circuit
•Sunload sensor
•HVAC control module

DTC Confirmation Procedure
1)Start engine and operate A/C system.
DTC Troubleshooting
NOTE:
•When DTC B1502-H is indicated together, it is possible that sensor ground circuit is open.
•DTC B1504 is detected even though there is not any malfunction if sunload sensor is not lit by an incandescent light. Light the sunload sensor with a light.
