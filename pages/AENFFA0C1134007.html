Scan Tool Data
ECM
As the data values below are standard values estimated on the basis of values obtained from the normally operating vehicles using a SUZUKI scan tool, use them as reference values. Even when the vehicle is in good condition, there may be cases where the checked value does not fall within each specified data range. Therefore, judgment as abnormal should not be made depending on these data alone. Also, conditions that can be checked by SUZUKI scan tool are those detected by ECM and output from ECM as commands and there may be cases where the engine or actuator is not operating (in the condition) as indicated by SUZUKI scan tool.
NOTE:
•If communication between SUZUKI scan tool and ECM is not possible, perform the following checks:
—Check communication condition between PC and VCI using SUZUKI scan tool function.
—Check DLC power circuit and ground circuit.
—Check DLC cable. If necessary, check DLC cable by substituting a known-good DLC cable.
—Check CAN communication system. (“Control Module Communication Bus Off” is detected in any control module.)  
—Check ECM power circuit and ground circuit.  
•When checking data with engine running at idle or racing, place gear shift lever in neutral position and apply parking brake fully.

Scan Tool Data Definitions
Engine Speed (rpm)
This parameter indicates revolutions per minute of engine.
Vehicle Speed (km/h, MPH)
This parameter indicates vehicle speed.
Odometer (km)
This parameter indicates odometer counts from ECM.
Battery Voltage (V)
This parameter indicates battery positive voltage inputted from main relay to ECM.
Accel Position (Accelerator pedal is fully released position, %)
Indication is 0 to 5% when accelerator pedal is in fully released position and 95 to 100% when the pedal is in fully depressed position.
APP Sensor 1 Volt (APP sensor (main) output voltage, V)
The Accelerator Pedal Position (APP) (Main) reading provides accelerator pedal opening information in voltage.
APP Sensor 2 Volt (APP sensor (sub) output voltage, V)
The Accelerator Pedal Position (APP) (Sub) reading provides accelerator pedal opening information in voltage.
ECT (Engine coolant temperature, °C, °F)
This parameter indicates engine coolant temperature measured by ECT sensor.
IAT1 (Intake Air Temperature-1, °C, °F)
This parameter indicates intake air temperature-1 measured by IAT sensor-1.
IAT2 (Intake Air Temperature-2, °C, °F)
This parameter indicates intake air temperature-2 measured by IAT sensor-2.
MAF (Mass air flow rate) (g/sec., Lb/min)
This parameter indicates mass air flow rate measured by MAF sensor.
Barometric pressure (kPa, mmHg, in.Hg, psi, inH2O)
This parameter indicates barometric air pressure measured by barometric pressure sensor in ECM.
MAP (kPa, mmHg, in.Hg, psi, inH2O)
This parameter indicates boost pressure measured by boost pressure sensor.
A/F Sensor Heater Duty (%)
This parameter indicates the ON time ratio of control signal for A/F sensor heater.
EGR-1 Valve Opening (%)
This parameter indicates the valve ON (valve open) time ratio EGR-1 valve controls signal.
Differential Pressure (kPa, mmHg, in.Hg, psi, inH2O)
This parameter indicates differential pressure between inlet and outlet of DPF® as measured by DPF® differential pressure sensor.
Inlet DPF gas Temp (Exhaust gas temperature, °C, °F)
This parameter indicates EGT at inlet of DPF® as measured by EGT sensor-2.
DPF Flow Resistance (hPa/m3/h)
This parameter indicates flow resistance of DPF®.
Soot Mass (%)
This parameter indicates how much of DPF® is clogged in percent.
Oil Life (%)
This parameter indicates the service life rate of engine oil.
EGR-2 Valve Opening (%)
This parameter indicates the valve ON (valve open) time ratio EGR-2 valve control signal.
Glow Plug Low Voltage (%)
This parameter indicates the ON time ratio of glow plug control module signal.
Battery Current (A)
This parameter indicates electric load value (current consumption) that is detected by electric load current sensor.
Battery Temperature (°C, °F)
This parameter indicates battery temperature measured by battery temperature sensor.
Commanded Alternator Regulated Voltage (V)
This parameter indicates commanded regulated-voltage to generator.
Fuel Rail Pressure (Fuel pressure in common rail, kPa, mmHg, in.Hg, psi, inH2O)
This parameter indicates fuel pressure in common rail measured by fuel pressure sensor.
Fuel temperature (°C, °F)
This parameter indicates fuel temperature measured by fuel temperature sensor.
Fuel Flow Actuator (%)
This parameter indicates duty ratio of the signal that controls opening time of fuel flow actuator.
Total Injected Fuel (mm3/str)
This parameter indicates total amount of fuel injected in 2 engine revolutions.
Inlet Catalyst gas Temp (Exhaust gas temperature, °C, °F)
This parameter indicates EGT at inlet of catalyst as measured by EGT sensor-1.
Outlet DPF gas Temp (Exhaust gas temperature, °C, °F)
This parameter indicates EGT at outlet of DPF as measured by EGT sensor-3.
Outlet EGR cooler gas Temp (Exhaust gas temperature, °C, °F)
This parameter indicates EGT at outlet of EGR cooler as measured by EGT sensor-4.
ENG A-STOP (N/A / Applied)
N/A: Selected vehicle variant data is “without ENG A-STOP system”.
Applied: Selected vehicle variant data is “with ENG A-STOP system”.
Adaptive Cruise Control (N/A / Applied)
N/A: Vehicle variant data is “without Adaptive Cruise Control” has been selected.
Applied: Vehicle variant data is “with Adaptive Cruise Control” has been selected.
Speed Limiter (N/A / Applied)
N/A: Vehicle variant data is “without Speed Limiter” has been selected.
Applied: Vehicle variant data is “with Speed Limiter” has been selected.
A/C Refrigerant Type (HFO1234yf / HFC134a)
This parameter indicates A/C refrigerant type.
Additional Heater (N/A / Applied)
N/A: Selected vehicle variant data is “without additional heater”.
Applied: Selected vehicle variant data is “with additional heater”.
Cruise (N/A / Applied)
N/A: Selected vehicle variant data is “without cruise control system”.
Applied: Selected vehicle variant data is “with cruise control system”.
A/C (N/A / Applied)
N/A: Selected vehicle variant data is “without A/C system”.
Applied: Selected vehicle variant data is “with A/C system”.
Additional Heater 1 (OFF / ON)
OFF: No command is being output.
ON: ON command is being output to additional heater relay No.1.
Additional Heater 2 and 3 (OFF / ON)
OFF: No command is being output.
ON: ON command is being output to additional heater relays No.2 and No.3.
A/C Switch (On/Off)
On: Command for A/C operation is being output from ECM to HVAC.
Off: Command for A/C operation is not being output.
Blower Fan (On/Off)
This parameter indicates the state of the blower speed selector.
Oil Pressure Switch 1 (ON / OFF)
This parameter indicates the state of the oil pressure switch-1.
Oil Pressure Switch 2 (ON / OFF)
This parameter indicates the state of the oil pressure switch-2.
Brake Switch (ON / OFF)
This parameter indicates the state of the brake light switch.
Cruise RES/ACC Switch (“RES / +” switch, OFF / ON)
OFF: “RES / +” switch is released.
ON: “RES / +” switch is pushed.
Cruise SET/COAST Switch (“SET / –” switch, OFF /ON)
OFF: “SET / –” switch is released.
ON: “SET / –” switch is pushed.
Cruise CANCEL Switch (“CANCEL” switch, OFF / ON)
OFF: “CANCEL” switch is released.
ON: “CANCEL” switch is pushed.
Cruise ON/OFF Switch (Released / Pushed)
Released: “CRUISE” switch is released.
Pushed: “CRUISE” switch is pushed.
MIL (ON / OFF / Check)
This parameter indicates operating state of MIL.
Distance fr DPF Light On (Distance from DPF® warning light ON, Km / mile)
This parameter indicates distance covered after DPF® warning light ON.
Distance fr DPF Replace (Mileage from DPF® is replaced, km/mile)
This parameter indicates distance covered after replacing DPF®.
Distance from Regeneration (Mileage from DPF® is regeneration, km/mile)
This parameter indicates distance covered after last after-sales regeneration of DPF®.
