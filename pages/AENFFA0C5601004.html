Select Lever Assembly Description
Select lever assembly (1) mainly consists of a straight type select lever (2), shift lock solenoid (inside the case) (3) and gear shift control module (4) communicating the shift information with TCM through CAN. P, R, N, D and M positions are available operating the select lever. However, shift changes from N to R, R to P and P to R require lock release operation by pressing the select lever button (detent button) (5). Each select lever position and shift lock solenoid functions as follows.
