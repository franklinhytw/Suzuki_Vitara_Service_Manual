Scan Tool Utility Table
SUZUKI scan tool has setting function for oil life monitoring system of combination meter as follows. For details of SUZUKI scan tool, refer to its Operator’s Manual.
Item Setting Description
Oil Change Request Light Configuration Oil Change Request Light Active / Disabled Oil change request light is selectable between “Active” and “Disabled”.
Oil Change Condition Normal / Severe / Optional Oil change interval is selectable between “Normal”, “Severe” and “Optional”.
Oil Change Distance Condition 2,500 – 30,000 km When oil change interval is set to “Optional”, oil change distance interval is selectable between 2,500 and 30,000km (1,500 and 18,000 miles) in 500 km (300 miles) increments.
Oil Change Period Condition 90 – 1,095 days When oil change interval is set to “Optional”, oil change period interval is selectable between 90 and 1,095 days in 5-day increments.
Oil Change Request Light Initialization — Cumulative distance and period can be reset.
