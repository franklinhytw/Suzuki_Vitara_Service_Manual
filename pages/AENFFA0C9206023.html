Rear Vehicle Height Sensor Removal and Installation (LED Headlight Model)
CAUTION:
Headlight auto leveling control module cannot adjust optical axis of headlights properly.
Do not remove bracket and link from rear vehicle height sensor.
NOTE:
After any of the following services, initialize headlight auto leveling system.  
•Replacement of the headlight auto leveling control module
•Replacement of the rear vehicle height sensor
•Removal of the rear vehicle height sensor
Removal
1)Set ignition “OFF”.
2)Disconnect rear vehicle height sensor connector (1) and clamp (2).
3)Remove rear vehicle height sensor bracket bolts (3) and then remove rear vehicle height sensor (4).
Installation
Reverse removal procedure noting the following points.
•Tighten rear vehicle height sensor bracket bolts to specified torque in numerical order (“1” – “3”).
Tightening torque
Rear vehicle height sensor bracket bolt (a): 10 N·m (1.0 kgf-m, 7.5 lbf-ft)
