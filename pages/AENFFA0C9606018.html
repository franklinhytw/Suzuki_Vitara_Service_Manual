9F
Ignition Key Registration (Ignition Switch Model)
NOTE:
Refer to Operator’s Manual of SUZUKI scan tool for the procedure of immobilizer registration.
•Register ignition key ID code using “Immobilizer Key Registration” function in SUZUKI scan tool.
•Ignition key registration is performed through “Immobilizer Key Registration”, and this registers the immobilizer transponder code.
•Performing “Immobilizer Key Registration” clears all of the ID codes already registered. Prepare all of the ignition keys.
•It is possible to register 4 ignition keys in keyless entry control module.
•If the keyless entry control module is replaced, register all of the ignition keys.
