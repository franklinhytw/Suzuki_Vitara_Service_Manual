9F
Keyless Entry Control Module Self-Diagnosis Function
Description
Keyless entry control module monitors conditions of the system components and its circuit. When an abnormal condition in the system occurs, keyless entry control module performs control as follows.
Self-Diagnosis Function
Keyless entry control module is equipped with self-diagnosis function that stores DTC for abnormalities detected in keyless entry system and immobilizer control system. Stored DTC is saved until abnormal condition is corrected.
Warning Function
•When a malfunction of keyless entry system and immobilizer control system is detected, keyless entry control module turns on theft deterrent light and/or immobilizer indicator light to warn the driver.
•After detecting a malfunction, theft deterrent light and immobilizer indicator light are turned on. However they are turned off when normal condition is detected.
