5F
Accumulator Removal and Installation
Reference: Solenoid Valve Unit Components
Removal
1)Perform “Hydraulic Circuit Depressurization” function under “Utility” mode on SUZUKI scan tool referring Hydraulic Circuit Depressurization Procedure.
2)Disconnect negative (–) cable at battery.
3)Hoist vehicle and remove engine undercover.
4)Disconnect electric motor connector (1).
5)Loosen accumulator band bolt (1).
6)Remove hydraulic system filter oil pipe clamp and detach hydraulic system filter oil pipe.
7)Loosen flare nuts (2) using special tool and remove accumulator pipe (3).
Special Tool
09950–78230
8)Remove accumulator (4).
Installation
Reverse removal procedure noting the following points.
•Tighten accumulator (1) to specified torque.
Tightening torque
Accumulator (a): 110 N·m (11.2 kgf-m, 81.5 lbf-ft)
•Tighten accumulator pipe flare nuts (2) to specified torque.
Tightening torque
Accumulator pipe flare nut (b): 18 N·m (1.8 kgf-m, 13.5 lbf-ft)
•Tighten accumulator band bolt (3) securely.
•Fix hydraulic system filter oil pipe with new clamp securely.
•After replacing accumulator, perform TCM and Clutch Initialization.
