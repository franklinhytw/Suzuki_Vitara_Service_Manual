Select Lever Assembly Removal and Installation
Removal
1)Disconnect negative (–) cable at battery.
2)Remove floor console front cover.  
3)Disconnect cable end holder from select lever assembly and then detach select cable from bracket.
4)Disconnect select lever connector.
5)Remove select lever nuts and detach select lever assembly from floor panel.
Installation
Reverse removal procedure noting the following points.
•Tighten select lever nuts to specified torque.
Tightening torque
Select lever nut (a): 13 N·m (1.3 kgf-m, 9.5 lbf-ft)
•After installing select lever assembly, adjust parking cable.  
•Check that ignition can be turned from ON to OFF when releasing select lever button in P range and that ignition cannot be turned from ON to OFF when releasing select lever button in N range.
•Check select lever operation.
