Select Lever Assembly Removal and Installation
Reference: Select Lever Assembly Components
Removal
1)Disconnect negative (–) cable at battery.
2)Remove floor console front cover.  
3)Disconnect select cable (1) from select lever assembly (2) as shown in figure.
3. Select lever bracket 4. Select lever pin
4)For ignition switch model, disconnect key interlock cable end and detach cable casing cap from select lever assembly.  
5)Disconnect select lever connector.
6)Remove select lever nuts (1), and then remove select lever assembly (2) from floor panel.
Installation
Reverse removal procedure noting the following points.
•Tighten select lever nuts to specified torque.  
•After installing select lever assembly, adjust select cable.  
•After installing select lever assembly, check that ignition can be turned from “ON” to “OFF” when releasing select lever button in P range and that ignition cannot be turned from “ON” to “OFF” when releasing select lever button in N range.
•After installing select lever assembly, check select lever operation.
