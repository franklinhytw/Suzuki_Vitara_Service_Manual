9B
Front Fog Light Removal and Installation (Without DRL) (If Equipped)
NOTE:
•For replacement of front fog light bulb, remove front fender lining.
•After any of the following services, aim fog lights.  
—Front fog light bulb replacement
—Front fog light removal and installation
1. Front fog light bulb
Removal
1)Remove front bumper.  
2)Disconnect connector(s) from front fog light.
3)Remove fog light screws (1) and then remove front fog light (2).
4)Turn socket (3) counterclockwise to remove bulb, if necessary.
5)Remove LED DRL (4) (if equipped).
6)Remove front fog light bracket (5), if necessary.
Installation
Reverse removal procedure noting the following point.
•Install front fog light bracket screws in numerical order (“1” – “5”).
