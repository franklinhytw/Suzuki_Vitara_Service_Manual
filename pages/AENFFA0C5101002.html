A/T Diagnosis General Description
This vehicle is equipped with TCM, which controls the automatic shift up and shift down timing, TCC operation, etc. suitably to vehicle driving conditions.
TCM has an OBD system which detects a malfunction in this system.
When diagnosing a trouble in transaxle including this system, have full understanding of the outline of OBD System Description and each item in Precautions for Diagnosing Trouble and execute diagnosis according to A/T Control System Check to obtain correct result smoothly.
