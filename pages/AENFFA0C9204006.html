Scan Tool Data Definitions
Battery Voltage (V)
This parameter indicates battery voltage inputted from IG1 relay to headlight auto leveling control module.
Leveling Actuator (%)
This parameter indicates in percent how much correction is needed in headlight leveling actuators so that headlight auto leveling control module maintains a proper optical axis of headlights based on signals inputted from rear vehicle height sensor.
Rear Height Sensor (mV)
This parameter indicates the input signal from rear vehicle height sensor.
Vehicle Speed (km/h, MPH)
This parameter indicates the vehicle speed calculated ECM.
Headlight Switch (ON/OFF)
This indicates state of lighting switch.
•ON: Lighting switch is in ON or “AUTO” position and headlights are on.
•OFF: Lighting switch is in clearance light, “OFF” or “AUTO” position and headlights are off.
Right Side Headlight Circuit Open
This parameter indicates the state of the right side headlight circuit.
Left Side Headlight Circuit Open
This parameter indicates the state of the left side headlight circuit.
