5A
Time-Lag Test
This test is to check conditions of clutch, brake and fluid pressure. “Time-lag” means time elapsed since select lever is shifted with engine idling till shock is felt.
1)Connect SUZUKI scan tool to DLC.
2)Select “Data list” mode on scan tool.
3)Drive vehicle so that ATF temperature reaches normal operating temperature (50 to 80 °C (122 to 176 °F)).  
NOTE:
Engine coolant temperature rises when revving engine in P or N range, but ATF temperature does not rise in P or N range.
Drive vehicle to raise ATF temperature.
4)With chocks placed before and behind front and rear wheels respectively, depress brake pedal.
5)With a stopwatch ready, shift select lever from N to D range and measure time from that moment till shock is felt.
6)Similarly measure time-lag by shifting select lever from N to R range.
Gear shifting time-lag
N → D: Less than 1.0 sec.
N → R: Less than 1.5 sec.
NOTE:
•When repeating this test, wait at least one minute after select lever is shifted back to N range.
•Engine should be warmed up fully for this test.
•Repeat test 3 times and take average of those data for final time-lag data.
