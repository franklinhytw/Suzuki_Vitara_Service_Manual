7B
Precautions for Diagnosing Trouble
•Do not disconnect connectors from HVAC control module, battery cable from battery, HVAC control module ground wire harness from body or main fuse before checking diagnostic information (diagnostic trouble codes) stored in HVAC control module memory.
•Diagnostic trouble codes stored in HVAC control module can be checked on SUZUKI scan tool. Before checking diagnostic trouble codes, read this manual and Operator's Manual of SUZUKI scan tool to know how to read diagnostic trouble codes.
•Read Warning for Air Bag before inspection.
