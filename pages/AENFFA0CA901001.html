ENG A-STOP System Description
The ENG A-STOP system performs the following controls to reduce emissions, fuel consumption, and idling noise during the vehicle stops. The system monitors the vehicle condition depending on signals from the relevant sensors and switches.
•Engine auto stop control
•Engine auto start control
The system indicates status of these controls to the driver by lighting or flashing the ENG A-STOP indicator light located in the combination meter.
Engine Auto Stop Control
When all of the following conditions are met, the ENG A-STOP control module sends an engine stop signal to the ECM via the CAN communication line. The ECM then stops the engine.
•The battery is sufficiently charged.
•The battery temperature is between 5 and 70 °C (between 41 and 158 °F).
•The vehicle speed is below 0 km/h (0 mile/h).
•The vehicle speed higher than 6 km/h (4 mile/h) was recorded after starting off.
•The vehicle is not driven backward.
•The engine coolant temperature is between 18 and 95 °C (between 64 and 203 °F) (D16AA model).
•The engine coolant temperature is between 57 and 105 °C (between 135 and 221 °F) (M16A model / K14C model).
•The engine exhaust gas temperature is higher than the specified level (D16AA model).
•Except when DPF® is in regeneration or at least 5 seconds have passed since the end of DPF® regeneration (D16AA model). For details, refer to DPF® Regeneration Description:D16AA.
•The vacuum in the brake booster is higher than the specified level.
•The gear is shifted to “Neutral”. (M/T model)
•Clutch pedal is released. (M/T model)
•The select lever is shifted to D or N range. (A/T model / twin clutch system model)
•The slope angle is between –14% and 14% (A/T model / twin clutch system model)
•The brake pedal is depressed. (A/T model / twin clutch system model)
•The hydraulic pressure in the brake master cylinder is higher than the specified level. (A/T model / twin clutch system model)
•Accelerator pedal is released.
•Engine auto stop control is not inhibited by ENG A-STOP OFF switch.
•The hood is closed.
•The driver’s door is closed and the driver’s seat belt is fastened.
•The ECM has detected no DTC related to ENG A-STOP system.
•The ECM permits engine to stop.
•The ESP® control module has detected no DTC related to ENG A-STOP system.
•The ABS is not in operation.
•The cabin temperature is within the preset temperature range (Auto A/C model).
•Air outlet is controlled excluding “DEF” position (Auto A/C model).
•The ENG A-STOP control module has detected no DTC.
When the engine auto stop control is activated, an ENG A-STOP indicator light signal is sent to the combination meter via the CAN communication line. The ENG A-STOP indicator light then lights up.
NOTE:
If engine is started while the hood is open in case of jump starting etc., “engine auto stop control” is suspended until ignition is set to “OFF”.
Engine Auto Start Control
When the gear is in neutral (M/T model) and one or more of the following conditions are met, the ENG A-STOP control module sends a signal to the ECM via the CAN communication line to authorize start of the engine. This turns on the ST2 relay, which causes the starting motor to operate and the engine to be restarted.
•The ENG A-STOP OFF switch is pushed.
•Clutch pedal is depressed. (M/T model)
•Select lever is shifted to other than D or N range. (A/T model / twin clutch system model)
•At least 2 minutes have passed after automatic stop of the engine. (*1)
•Select lever is being shifted from N to D range. (A/T model / twin clutch system model)
•Brake pedal is released. (A/T model / twin clutch system model)
•Accelerator pedal is depressed. (A/T model / twin clutch system model)
•The hydraulic pressure in the brake master cylinder is lower than the specified level. (A/T model / twin clutch system model)
•Driver’s door is opened. (A/T model / twin clutch system model)
•Driver’s seat belt is unfastened. (A/T model / twin clutch system model)
•The vehicle speed is 4 km/h (3 mile/h) or higher. (*1)
•Battery discharge is more than specified level. (*1)
•Battery voltage is lower than specified value. (*1)
•The vacuum in the brake booster is lower than the specified level. (*1)
•ENG A-STOP system request signal is received from HVAC control module (Auto A/C model). (*1)
•The ENG A-STOP control module has detected specific DTC(s). This is an engine starting by fail-safe mode. For details, refer to Fail-Safe Table. (*1)
When the engine auto start control is activated, an ENG A-STOP indicator light signal is sent to the combination meter via the CAN communication line. And then the ENG A-STOP indicator light is turned off.
NOTE:
*1: The ENG A-STOP indicator light then flashes 2 times.
Engine Auto Start Inhibit Control
If any of the following conditions is met during the “engine auto stop control”, the ENG A-STOP control module suspends the “engine auto start control”.
•The ENG A-STOP control module has detected specific DTC(s). This is an engine start inhibit by fail-safe mode. For details, refer to Fail-Safe Table.
•The hood is open.
•The driver’s door is open and the driver’s seat belt is unfastened. (M/T model)
NOTE:
Once the “engine auto start control” is inhibited, driver’s operation is needed to restart the engine.
Engine Restart Control after Engine Stall (M/T Model)
While driving vehicle, if the engine stalled caused by a clutch operation error etc., the engine will restart by depressing the clutch pedal without turning the ignition key to “START” (ignition switch model) or without pushing the engine switch to start (keyless push start model) when the following conditions are met.
•The gear shift lever is “Neutral”.
•The clutch pedal is depressed fully.
NOTE:
This auto restart control may not work if the engine coolant temperature does not meet restart conditions such as low temperature. In this case, the engine can be restarted in the conventional manner.
