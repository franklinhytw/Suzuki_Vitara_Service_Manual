DTC Table
NOTE:
•If you do not perform P/S System Check before starting the DTC diagnosis, the diagnosis will show a false result.
Perform P/S System Check before starting the diagnosis.
•*D/C (Driving Cycle) is a duration from engine start to the following engine shutoff.
—1 D/C: When a malfunction is detected in 1st driving cycle, P/S warning light is turned ON.
•*1: Whether P/S warning light lights up or not depends on DTC detecting condition.
DTC DTC name DTC detecting condition *D/C P/S warning light
C1111   Torque Sensor (Main) Circuit Malfunction Voltage of torque sensor signal (main) circuit is higher than specified value (4.5 V) or lower than specified value (0.5 V) for more than specified time. 1 Lights up
C1113   Torque Sensor (Main and Sub) Signal Difference Voltage difference between torque sensor signal (main) circuit and torque sensor signal (sub) circuit remains too large for more than specified time. 1 Lights up
C1115   Torque Sensor (Sub) Circuit Malfunction Voltage of torque sensor signal (sub) circuit is higher than specified value (4.5 V) or lower than specified value (0.5 V) for more than specified time. 1 Lights up
C1116   Torque Sensor Power Supply (9 V) Circuit Malfunction Reference power supply voltage (9 V) for torque sensor circuit is lower than specified lower limit (7.0 V) for more than specified time. 1 Lights up
C1131   Rotation Angle Sensor Signal Malfunction Rotation angle sensor output signal error continues for more than specified period. 1 Lights up
C1141   P/S Motor Circuit Voltage Malfunction Measured circuit voltage is higher than standard voltage by 0.3 V or more or motor drive circuit is defective. 1 Lights up
C1142   P/S Motor Circuit Current Malfunction 1 Measured motor drive current is bigger than target value by 10 A or more. 1 Lights up
C1143   P/S Motor Circuit Current Too Big Measured motor drive current is more than 97 A. 1 Lights up
C1145   P/S Motor Circuit Current Malfunction 2 Target motor drive current is more than 4 A or measured value is less than 2 A. 1 Lights up
C1146   P/S Motor Voltage Malfunction Motor voltage calculated in P/S control module is abnormal. 1 Lights up
C1152   Fail-Safe Relay Malfunction (Melting Stuck) Measured voltage of fail-safe relay contact is higher than 5.5 V even though fail-safe relay-off is commanded. 1 Lights up
C1153   Power Supply Voltage Too Low Battery voltage is lower than 9 V even if one of the following conditions is met:
•Voltage at fail-safe relay contact is higher than 19.2 V.
•Engine speed is more than 600 rpm and fail-safe relay ON command is outputted.
 1 Lights up
C1154   Fail-Safe Relay Malfunction (Contact) Measured voltage of fail-safe relay contact is lower than 4 V even though fail-safe relay-on is commanded. 1 Lights up
C1155   P/S Control Module Internal Failure / Power Supply Voltage Too High P/S control module internal failure. *1 Lights up
C1160   Assist Map Signal Failure Assist map data is not set in P/S control module. 1 Lights up
C1161   CAN Invalid Data from Vehicle Speed Invalid data is received from ECM. 1 Off
C1162   CAN Invalid Data from Engine Speed Invalid data is received from ECM. 1 Lights up
C1164   CAN Invalid Data from BCM Invalid data about steering assist torque request is received from BCM or steering assist torque request from BCM is received for more than 30 sec. 1 Off
C1166   Reliability Defect Of Vehicle Speed Data Invalid data about vehicle speed reliability is received from ECM. 1 Off
U0073   Control Module Communication Bus Off Transmitting and receiving error of P/S control module for specified time continuously. 1 Off
U0100   Lost Communication With ECM CAN communication data from ECM cannot be received. 1 Lights up
U0121   Lost Communication With ABS/ESP® Control Module CAN communication data from ABS/ESP® cannot be received. 1 Lights up
U0140   Lost Communication With Body Control Module CAN communication data from BCM cannot be received. 1 Lights up
U0155   Lost Communication With Instrument Panel Cluster (IPC) Control Module CAN communication data cannot be received from combination meter. 1 Lights up
U1082   Lost Communication With ENG A-STOP Control Module CAN communication data cannot be received from ENG A-STOP control module. 1 Lights up
