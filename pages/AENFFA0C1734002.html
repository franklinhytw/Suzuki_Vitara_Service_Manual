1G
High-Pressure Fuel Supply Circuit Check
Before performing the following procedure, observe Precautions for Fuel System Service:D16AA.
Troubleshooting
NOTE:
•Before troubleshooting, check low-pressure fuel circuit.  
•Before troubleshooting, check that battery voltage is 11 V or more.
Step Action Yes No
1
Fuel injector calibration code check
1)Using SUZUKI scan tool, check that fuel injector calibration codes registered in ECM are corresponded with calibration codes stamped on fuel injectors.
Is check result OK?
Go to Step 2.
Register correct fuel injector calibration codes in ECM.  
2
Fuel injector check
1)Check fuel injector for resistance.  
Is check result OK?
Go to Step 3.
Replace faulty fuel injector.  
3
Fuel injector leakage check
1)Relieve fuel pressure.  
2)Set ignition “OFF”.
3)Disconnect connectors from all injectors.
4)After removing clips (2), disconnect fuel return hose (1) from all fuel injectors.
5)Plug fuel return hose with plug cap so that no fuel comes out during this check. For details, refer to Precautions for Fuel System Service:D16AA.
6)Connect special tool to fuel injectors.
NOTE:
For adapters between special tool and injectors, use return fuel hose supplied as spare part.
Special Tool
(A): 09912–96540
7)Crank engine for 5 seconds.
8)Check that quantities of return fuel from all injectors are almost the same.
Is check result OK?
Go to Step 4.
Replace faulty fuel injector.  
4
Fuel injector leakage check
1)Set ignition “OFF”.
2)Connect connectors to all injectors.
3)Check that quantities of return fuel from all injectors are almost the same under each of the following condition respectively in the same manner as in Step 3.
•Engine running at idle speed for 3 min.
•Engine running at 1,500 rpm for 3 min.
•Engine running at 3,000 rpm for 3 min.
Is check result OK?
Go to Step 5.
Replace faulty fuel injector.  
5
High pressure fuel circuit check
1)Set ignition “OFF”.
2)Disconnect connector from all fuel injectors.
3)Connect SUZUKI scan tool to DLC.
4)Crank engine for 5 seconds.
5)Check that “Fuel Rail Pressure” displayed on SUZUKI scan tool is higher than 30 MPa (300 bar) with engine running at 200 rpm or more.
Is check result OK?
End.
Go to Step 6.
6
Fuel pressure sensor circuit check
1)Check fuel pressure sensor circuit.  
Is check result OK?
Go to Step 7.
Repair or replace defective wire harness.
7
Fuel flow actuator operation check
1)Check fuel flow actuator operation.  
Is check result OK?
End.
Repair or replace defective part.
