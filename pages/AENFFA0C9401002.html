Rain-Sensing Wiper System Description (If Equipped)
•With the wiper and washer switch in the “AUTO” position, the rain-sensing wiper system automatically turns on the windshield wiper motor when the rain / light sensor senses waterdrops on the windshield. The system also controls the wiping speed and intermittent operation interval of the wipers according to both rain / snow condition and vehicle speed.
•The rain-sensing wiper system is controlled by BCM.
•Turning the AUTO knob on the wiper and washer switch adjusts the rain / light sensor’s sensitivity to waterdrops on the windshield.
•The rain-sensing wiper system can be set to any of the following three operating modes. (Refer to Rain / Light Sensor Setting Change Procedure (If Equipped).)
—Manual intermittent wiping mode
—Semi-automatic wiping mode
—Full-automatic wiping mode
