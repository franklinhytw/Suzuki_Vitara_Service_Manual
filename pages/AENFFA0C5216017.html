5B
Transaxle Right Case Disassembly and Reassembly
Reference: Manual Transaxle Unit Dismounting and Remounting:5 M/T
Reference: Manual Transaxle Assembly Components:5 M/T
Disassembly
1)For 2WD model, remove differential side oil seal.  
2)Remove differential side bearing outer race (1) using brass bar (2), if necessary.
3)Remove input shaft oil seal (1), if necessary.
4)Pull out countershaft right bearing outer race using special tools, if necessary.
Special Tool
(A): 09930–30104
(B): 09941–64511
Reassembly
1)Install input shaft oil seal (1) facing its spring side upward until it becomes flush with case surface using special tool and hammer. Apply grease to oil seal lip.
: Grease 99000–25011 (SUZUKI Super Grease A)
Special Tool
(A): 09913–84510
2)Install countershaft right bearing outer race (1) using special tools and hammer.
Special Tool
(A): 09913–75821
(B): 09924–84510–004
3)Install differential side bearing outer race using special tool and hammer.
Special Tool
(A): 09924–74510
(B): 09925–14520
4)For 2WD model, install differential side oil seal.
