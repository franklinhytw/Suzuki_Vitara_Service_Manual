Rear Wheel Speed Sensor Encoder Removal and Installation
2WD Model
NOTE:
Rear wheel speed sensor encoder is included in rear wheel hub assembly. If rear wheel speed sensor encoder needs to be replaced, replace rear wheel hub assembly.
For removal and installation of rear wheel speed sensor encoder (included in rear wheel hub assembly), refer to Rear Wheel Hub Assembly Removal and Installation:Rear.
4WD Model
NOTE:
Rear wheel speed sensor encoder is included in rear wheel hub assembly. If rear wheel speed sensor encoder needs to be replaced, replace it as a rear wheel hub assembly.
Refer to Rear Wheel Hub Assembly Removal and Installation:Rear.
