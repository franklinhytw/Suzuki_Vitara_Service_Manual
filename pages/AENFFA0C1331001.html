Description of ECM Registration
The following vehicle information is registered in ECM.
•Injector calibration code
•Vehicle variant data (additional heater, cruise control, A/C)
•ECM substitution data (oil life, soot mass, mean value, number of time)
•Differential pressure sensor data
•MAF sensor data
•Engine oil parameter
•EGR-1 data
•EGR-2 data
•DPF® driving pattern parameter
•A/F sensor dew point
•Fuel injection system part parameter
•A/F sensor parameter
•DPF® regeneration parameter
•DPF® condition parameter
•Driven distance with fuel filter warning light ON
•Rail pressure sensor data
•Catalyst condition parameter
•Immobilizer ID code
