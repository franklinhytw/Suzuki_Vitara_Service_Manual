Scan Tool Data Definitions
Accel Position (Accelerator pedal position) (%)
This parameter indicates accelerator pedal opening ratio detected by signal on CAN communication line fed from ECM.
Transfer Oil Temperature (°C, °F)
This parameter indicates transfer oil temperature detected by transfer oil temperature sensor installed in transfer assembly.
4WD Control Drive CUR. (4WD control drive current) (A)
This parameter indicates input current of rear drive coupling unit.
Battery Voltage (V)
This parameter indicates battery voltage detected by 4WD control module.
4WD Unit Temperature (°C, °F)
This parameter indicates rear drive coupling unit temperature detected by rear drive coupling air temperature sensor installed in rear drive coupling assembly.
Front Wheelspeed, Rear Wheelspeed (km/h, MPH)
Wheel speed is an ESP® control module internal parameter. It is computed by reference pulses from the wheel speed sensor.
F-R Wheelspeed Difference (Front-rear wheel speed difference) (rpm)
This parameter indicates rotation difference between front wheel and rear wheel detected by 4WD control module.
4WD Control Duty Ratio (%)
This parameter indicates operation rate of rear drive coupling unit.
Number of Installation Times of Different Diameter Tire
This parameter indicates number of detection times of different diameter tire(s) detected by 4WD control module.
Number of Times of Coupling Unit Abnormal Heat Generation
This parameter indicates number of detection times of abnormal heat generation detected by 4WD control module.
Brake Switch
This parameter indicates ON/OFF of brake light switch sent from ECM through CAN.
Coupling Unit Power Transistor
This parameter indicates state of power supply to rear drive coupling unit.
ABS Control Status
This parameter indicates ABS status sent from ESP® control module through CAN.
TCS Control Status
This parameter indicates traction control status sent from ESP® control module through CAN.
Stability Control Status
This parameter indicates stability control status sent from ESP® control module through CAN.
4WD Warning Light
This parameter indicates the states of 4WD warning light output signal.
