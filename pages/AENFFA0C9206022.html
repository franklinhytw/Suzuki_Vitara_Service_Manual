9B
Headlight Auto Leveling Control Module Removal and Installation (LED Headlight Model)
NOTE:
After any of the following services, initialize headlight auto leveling system.  
•Replacement of the headlight auto leveling control module
•Replacement of the rear vehicle height sensor
•Removal of the rear vehicle height sensor
Removal
1)Set ignition “OFF”.
2)For LHD, remove drive side instrument panel undercover.
For RHD, remove glove box.
3)Disconnect headlight auto leveling control module connector.
4)Remove nut (1) and then remove headlight auto leveling control module (2).
Installation
Reverse removal procedure noting the following point.
•Tighten headlight auto leveling control module nut to specified torque.
Tightening torque
Headlight auto leveling control module nut (a): 9.0 N·m (0.92 kgf-m, 7.0 lbf-ft)
