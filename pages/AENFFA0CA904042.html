10I
Starting Motor Operation Inspection
NOTE:
Check that battery voltage is 12 V or more.
1)Check that ignition is “OFF”.
2)Connect SUZUKI scan tool to DLC.
3)For M/T model, place gear shift lever in “Neutral” and apply parking brake.
For A/T model, place select lever in P range and apply parking brake.
For twin clutch system model, place select lever in P range and apply parking brake.
4)Set ignition “ON”.
5)Select “Starting Motor” of “Active Test” mode on SUZUKI scan tool.
6)Check that starting motor operates properly.
If any faulty condition is found, check the following points.
•Whether any DTC is stored or not
•ST2 relay harness
•Starting motor harness
•Starting motor
