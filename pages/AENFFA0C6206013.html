Steering Lock Inspection (Ignition Switch Model)
NOTE:
Steering lock can be checked on vehicle.
1)Remove steering column lower cover.  
2)Disconnect key reminder switch connector from steering lock.
Check continuity between terminals of key reminder switch. If check result is not as specified, replace steering lock.
Key reminder switch continuity
[A]: Terminal [C]: Key inserted
[B]: Condition [D]: Key pulled out
3)Connect connector to key reminder switch.
4)Install steering column lower cover.
