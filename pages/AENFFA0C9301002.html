Parking Sensor System Description (If Equipped)
•The parking sensor system consists of the parking sensor system control module, combination meter (information display), front parking sensors (if equipped), rear parking sensors, parking sensor front buzzer (if equipped), parking sensor rear buzzer, parking sensor OFF switch, back up light switch (M/T model), and back up light relay (A/T model and twin clutch system model).
•The parking sensor system activates illuminating indicator light when the parking sensor OFF switch is pressed (ON) and:
—The shift position is in “N”, “D”, or “M” range (A/T model and twin clutch system model) with the vehicle speed approx. 10 km/h (6 mile/h) or lower.
—The shift position is in “R” range.
•If the parking sensors located on the front and rear bumpers detect an obstacle, the parking sensor system control module causes a symbol representing the direction of and distance to the obstacle to be indicated on the information display, in addition to causing the parking sensor buzzer to sound.
•The parking sensor system control module communicates with the combination meter via the LIN communication system. (For LIN communication, refer to Communication System Description:M16A Model or Communication System Description:D16AA Model or Communication System Description:K14C Model.)
•DTC for parking sensor control system can be checked by selecting the “Combination Meter / Meter/M&A” on SUZUKI scan tool.
