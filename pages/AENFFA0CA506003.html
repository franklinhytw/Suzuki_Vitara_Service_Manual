Keyless Start Control Module Registration
After keyless start control module is replaced with new one, register remote controller ID code in keyless start control module using SUZUKI scan tool and security device.
1)Prepare all of remote controllers ready for registration.
2)Perform “Immobilizer Key Registration” in “Immobilizer” menu on SUZUKI scan tool. Refer to Operator’s Manual of SUZUKI scan tool for further details.
NOTE:
When ECM and keyless start control module are replaced with new ones at the same time, perform “Immobilizer Key Registration” instead of “ECM Registration” using SUZUKI scan tool and security device. Refer to Operator’s Manual of SUZUKI scan tool for the immobilizer registration.
