Electronic Control System Description
The electronic control system consists of various sensors which detect the state of engine and driving conditions, ECM which controls various devices according to the signals from the sensors and various controlled devices.
Functionally, the electronic control system is divided into the following sub systems:
•Fuel injection control system
•Ignition control system
•Electric throttle control system
•Fuel pump control system
•Radiator cooling fan control system
•Evaporative emission control system
•HO2S heater control system
•A/C control system
•VVT system
•Generator control system
•Boost control system
•Immobilizer control system
•Controller (computer) communication system
ECM is uses CAN communication system to communicate with other control modules. For more detail of CAN communication, refer to Communication System Description:K14C Model.
