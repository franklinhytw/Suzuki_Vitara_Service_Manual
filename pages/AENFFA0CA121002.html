10A
Description of Adaptive Cruise Control System Components
Component Description
RBS control module RBS control module incorporates a radar sensor and is located in the radiator grill. This control module determines the vehicle to follow and target level of acceleration / deceleration based on information obtained by millimeter wave radar, and demands brake control and torque control for adequate distance to a vehicle ahead.
For more details of RBS control module, refer to Description of RBS System Components.
CRUISE (main) switch
•This switch has a momentary contact type button to turn adaptive cruise control function ON and OFF.
•The switch signal is outputted to ECM and RBS control module receives cruise control switch signal from ECM through CAN communication.
Distance switch
•Vehicle distance under “Constant headway gap control” can be changed to “Long”, “Middle” or “Short” mode using this switch.
•The switch signal is outputted to ECM and RBS control module receives cruise control switch signal from ECM through CAN communication.
SET/– switch
•When this switch is pressed (ON) and then released (OFF) while the vehicle is running at 40 km/h (25 mile/h) or higher speed, the vehicle speed obtained when the switch is released is stored in memory and it is set as a target vehicle speed under adaptive cruise control.
•When this switch is pressed (ON) for a momentarily during adaptive cruise control, the set speed will decrease by 1 km/h (1 mile/h). When this switch is continuously pressed (ON) during adaptive cruise control, the set speed will decrease by 5 km/h (5 mile/h) as long as the switch is ON. And the vehicle is controlled according to the new set speed.
•The switch signal is outputted to ECM and RBS control module receives cruise control switch signal from ECM through CAN communication.
RES/+ switch
•When this switch is pressed (ON) for a momentarily during adaptive cruise control, the set speed will increase by 1 km/h (1 mile/h). When this switch is continuously pressed (ON) during adaptive cruise control, the set speed will increase by 5 km/h (5 mile/h) as long as the switch is ON. And the vehicle is controlled according to the new set speed.
•If vehicle speed is higher than 40 km/h (25 mile/h) after adaptive cruise control is canceled, pressing this switch ON momentarily will resume the vehicle running at the speed.
•The switch signal is outputted to ECM and RBS control module receives cruise control switch signal from ECM through CAN communication.
CANCEL switch
•When this switch is pressed (ON), adaptive cruise control is canceled.
•The switch signal is outputted to ECM and RBS control module receives cruise control switch signal from ECM through CAN communication.
ECM
•ECM controls electric throttle valve opening (M16A model / K14C model) or fuel injector (D16AA model) based on information from RBS control module through CAN communication.
•ECM transmits cruise control switch signal, engine torque signal, vehicle speed signal, etc. to RBS control module through CAN communication.
ESP® control module
•ESP® control module controls brake pressure and brake light relay based on information from RBS control module through CAN communication.
•ESP® control module transmits wheel speed signal, lateral G signal, ABS/ESP® status, etc. to RBS control module through CAN communication.
Steering angle sensor The sensor transmits steering angle signal to RBS control module through CAN communication.
Combination meter Combination meter turns on or off adaptive cruise control warning light, shows state of the control on information display and sounds warning buzzer based on information through CAN communication.
ENG A-STOP control module ENG A-STOP control module transmits neutral switch status to RBS control module through CAN communication.
TCM (A/T model / twin clutch system model) TCM transmits transmission shift position signal to RBS control module through CAN communication.
BCM
•BCM transmits turn signal switch status etc. to RBS control module through CAN communication.
•RBS control module uses turn signal switch status to judge overtaking action.
