NOTE:
•Perform cruise control / speed limiter system check referring to Cruise Control / Speed Limiter System Check:Cruise Control before performing this Cruise Control / Speed Limiter System Symptom Diagnosis.
•Check each part in the order from the top of the following list.
Condition Possible Cause Action
