TPMS Control Module Removal and Installation
Removal
1)Disconnect negative (–) cable at battery.
2)Remove center console box.  
3)Remove mid inside antenna (if equipped).  
4)Disconnect TPMS control module connector (1).
5)Remove screw (3), and then remove TPMS control module (2).
Installation
Reverse removal procedure noting the following point.
•If TPMS control module is replaced, perform Registration Procedure for Tire Pressure Sensor ID Code.
