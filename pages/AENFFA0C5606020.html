Parking Cable Adjustment
Reference: Parking Cable Removal and Installation
Reference: Parking Cable Inspection
1)Apply parking brake and block wheels.
2)Set ignition “ON” and shift select lever to P range.
3)Remove floor console front cover.  
4)Release lock plate (1) which restricts moving of cable end holder (2).
5)Push out cable end holder from adjuster (4) using appropriate tool (3) to disengage cable.
6)Shift select lever to P range.
7)Apply grease to select lever pin (5), and then install adjuster (1) into select lever pin securely.
“A”: Grease 99000–25011 (SUZUKI Super Grease A)
8)Push cable end holder (2) in arrow direction [a] as shown in figure.
NOTE:
At this time, do not apply force to adjuster and cable end in the direction of [c].
9)Slide lock plate (3) in arrow direction [b] as shown in figure, until it is locked with the claw (4) of cable end holder.
10)After installing select cable, check the following points.
•Perform P Range Test.
•Check that back up light turns on when shifting select lever to R range.
•Check that shift position indicated in information display is consistent with select lever range.
•Check that engine starts in P and N ranges but does not start in ranges other than P and N.
11)Install floor console front cover.
