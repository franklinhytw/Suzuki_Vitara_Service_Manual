Electric Load Current Sensor On-Vehicle Inspection
Battery Current
Using SUZUKI scan tool
1)When ignition is “OFF”, connect SUZUKI scan tool to DLC.
2)Check that “Battery Current” displayed on SUZUKI scan tool changes to values as electric load increases.
•Non-ENG A-STOP model:  
•ENG A-STOP model:  
If check result is not as specified, check the following points.
•Electric load current sensor circuits (power, ground and output)
•Battery negative (–) cable
•The following charging system components
—Battery  
—Generator  
—Generator output circuit
—Generator LIN communication circuit according to Step 2 under DTC U2001:D16AA
If electric load current sensor circuits and charging system are in good condition, replace electric load current sensor (1).
Without using SUZUKI scan tool
1)When ignition is “OFF”, disconnect electric load current sensor connector.
2)Using service wire (1), connect electric load current sensor (2) and electric load current sensor connector (3).
3)With ignition “ON”, check that voltage varies depending on electric load.
•Non-ENG A-STOP model:  
•ENG A-STOP model:  
If check result is not as specified, check the following parts and circuit.
•Electric load current sensor circuits (power, ground and output)
•Battery negative (–) cable
•The following charging system components
—Battery  
—Generator  
—Generator output circuit
—Generator LIN communication circuit according to Step 2 under DTC U2001:D16AA
If electric load current sensor circuit and charging system are in good condition, replace electric load current sensor.
