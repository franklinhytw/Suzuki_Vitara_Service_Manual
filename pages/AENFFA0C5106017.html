5A
Select Cable Adjustment
Reference: Select Cable Removal and Installation
1)Apply parking brake and block wheels.
2)Set ignition “ON” and shift select lever to N range.
3)Remove floor console front cover.  
4)Remove adjuster case (cable end) from select lever pin of select lever assembly.  
5)Release lock plate (1) which restricts moving of lock piece (2).
6)Pull out lock piece (2) from adjuster case (3) to disengage cable.
7)Check that “N” position of select lever is in suitable position.
8)Shift select lever to N range.
9)Check that positioning protrusion (2) on manual select lever (1) is pointing to “N” position on TCM (3).
10)Apply grease to select lever pin, and then install select lever pin securely.
“A”: Grease 99000–25011 (SUZUKI Super Grease A)
11)Push lock piece (1) in arrow direction [a] as shown in figure.
NOTE:
At this time, do not apply force to adjuster and cable end in the direction of [c].
12)Slide lock plate (2) in arrow direction [b] as shown in figure, until it gets over the claw (3).
13)Check that claws (1) of lock piece protrude as shown.
14)After installing select cable, check the following points.
•Perform P Range Test.
•Check that back up light turns on when shifting select lever to R range.
•Check that shift position indicated on information display is consistent with select lever range.
•Check that engine starts in P and N ranges and does not start in other ranges.
15)Install floor console front cover.
