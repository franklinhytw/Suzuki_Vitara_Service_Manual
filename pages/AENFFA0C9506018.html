9E
Rear Window Regulator Assembly Removal and Installation (If Equipped)
Reference: Rear Door Window Glass Components
Removal
1)Remove rear door glass.  
2)For power window type, disconnect power window motor connector (1).
3)Remove window regulator bolts (2) and remove rear window regulator assembly (3).
Manual window type
Power window type
Installation
Reference: Rear Window Regulator Assembly Inspection (If Equipped)
Reverse removal procedure noting the following points.
•Apply grease to sliding parts of rear window regulator assembly.
: Grease 99000–25011 (SUZUKI Super Grease A)
•Tighten rear window regulator bolts in numerical order (“1” – “2”) to specified torque.
Tightening torque
Rear window regulator bolt* (a): 8.4 N·m (0.86 kgf-m, 6.5 lbf-ft)
Manual window type
Power window type
