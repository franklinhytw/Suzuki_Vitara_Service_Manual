Manual Release of Shift Lock
1)For ignition switch model, set ignition “ACC”.
2)For LHD model, open cap (1) at select lever assembly using a flat-bladed screwdriver.
3)While pushing down shift lock solenoid release plate (1) at inside of cap using a flat-bladed screwdriver (2), press the select lever button (3). Check that select lever (4) shifts to other than P range.
If faulty condition is found, replace select lever assembly.
