Scan Tool Data
TCM
As the data values given in the following table are standard values estimated on the basis of values obtained from the normally operating vehicles using a scan tool, use them as reference values. Even when the vehicle is in good condition, there may be cases where the checked value does not fall within each specified data range. Therefore, judgment as abnormal should not be made depending on these data alone.
Also, condition in the following table that can be checked by the scan tool are those detected by TCM and output from TCM as commands and there may be cases where the twin clutch system or actuator is not operating (in the condition) as indicated by the scan tool.
NOTE:
•If communication between SUZUKI scan tool and TCM is not possible, perform the following checks:
—Check communication condition between PC and VCI using SUZUKI scan tool function.
—Check DLC power circuit and ground circuit.
—Check DLC cable. If necessary, check DLC cable by substituting a known-good DLC cable.
—Check CAN communication system. (“Control Module Communication Bus Off” is detected in any control module.)  
—Check TCM power circuit and ground circuit.  
•The following scan tool data related to twin clutch system can be checked only by communicating with TCM.

Scan Tool Data Definitions
1st/3rd Engagement Actuator Position (mm)
This parameter indicates 1st & 3rd engagement actuator position detected by 1st & 3rd gear position sensor.
6th Engagement Actuator Position (mm)
This parameter indicates 6th engagement actuator position detected by 6th gear position sensor.
2nd/4th Engagement Actuator Position (mm)
This parameter indicates 2nd & 4th engagement actuator position detected by 2nd & 4th gear position sensor.
5th/R Engagement Actuator Position (mm)
This parameter indicates 5th & reverse engagement actuator position detected by 5th & reverse gear position sensor.
Selection Actuator Position (mm)
This parameter indicates gear shift actuator position detected by shift position sensor.
Odd Gears Clutch Actuator (K1) Position (mm)
This parameter indicates odd gear clutch actuator position detected by odd gear clutch position sensor.
Even Gears Clutch Actuator (K2) Pressure (bar)
This parameter indicates even gear clutch actuator pressure detected by even gear clutch pressure sensor.
Odd Gears Clutch Actuator (K1) Position Reference (mm)
Odd gear clutch actuator position reference value is detected by odd gear clutch position sensor.
Even Gears Clutch Actuator (K2) Pressure Reference (bar)
Even gear clutch actuator pressure reference value is detected by even gear clutch pressure sensor.
Accelerator Pedal Position (%)
This parameter indicates accelerator pedal position signal from ECM through CAN.
Hydraulic Circuit Pressure Sensor (bar)
This parameter indicates hydraulic circuit pressure detected by line pressure sensor.
Engine Water Temperature (°C, °F)
This parameter indicates engine coolant temperature signal from ECM through CAN.
External Temperature (Outside Air Temperature (°C, °F))
This parameter indicates outside air temperature signal from IPC through CAN.
Battery Voltage (V)
This parameter indicates battery positive voltage inputted to TCM.
Engine Speed (rpm)
This parameter indicates engine speed signal from ECM through CAN.
Odd Gears Clutch Disc (K1) Speed (rpm)
This parameter indicates odd gear clutch disc speed detected by odd gear clutch speed sensor.
Even Gears Clutch Disc (K2) Speed (rpm)
This parameter indicates even gear clutch disc speed detected by even gear clutch speed sensor.
Vehicle Speed (km/h)
This parameter indicates vehicle speed signal from ECM through CAN.
Engagement Electrovalve (PPV1) (Gear Shift Solenoid Valve No.1) Current (3rd/2nd/R) (mA)
This parameter indicates control current value that TCM outputs to gear shift solenoid valve No.1.
Engagement Electrovalve (PPV2) (Gear Shift Solenoid Valve No.2) Current (1st/6th/4th/5th) (mA)
This parameter indicates control current value that TCM outputs to gear shift solenoid valve No.2.
Odd Gears Clutch Electrovalve (QPV-K1) Current (mA)
This parameter indicates control current value that TCM outputs to odd gear clutch solenoid valve.
Even Gears Clutch Electrovalve (PPV-K2) Current (mA)
This parameter indicates control current value that TCM outputs to even gear clutch solenoid valve.
Ranges Selection Electrovalve (PPVS) Current (mA)
This parameter indicates control current value that TCM outputs to gear select solenoid valve.
Gear Lever Position
This parameter indicates select lever position signal from select lever through CAN.
Gear Engaged
This parameter indicates gear engaged position detected by each of gear position sensor.
Brake Status via CAN (ON/OFF)
This parameter indicates parking brake switch signal from BCM through CAN.
ON: Brake light switch is ON. (Brake pedal depressed)
OFF: Brake light switch is OFF. (Brake pedal released)
Door Wired (Driver Side Front Door Switch Wiring Harness) (ON/OFF)
This parameter indicates driver side front door switch wiring harness.
ON: Door is open.
OFF: Door is closed.
Brake Wired (Brake Light Switch Wiring Harness) (ON/OFF)
This parameter indicates brake light switch inputted to TCM.
ON: Brake light switch is ON. (Brake pedal depressed)
OFF: Brake light switch is OFF. (Brake pedal released)
Lever Block Solenoid Command (Shift Lock Solenoid) (ON/OFF)
This parameter indicates shift lock solenoid command signal from select lever through CAN.
ON: Brake pedal is released. (Select lever P)
OFF: Brake pedal is depressed. (Select lever P)
Cranking
This parameter indicates state of ST3 relay output.
ON: ST3 relay is ON.
OFF: ST3 relay is OFF.
Gear Request
This parameter indicates the states of gear request signal from select lever through CAN.
Sensor Module Temperature (°C, °F)
This parameter indicates sensor module temperature detected by sensor module temperature sensor.
Odd Gear Clutch (K1) Temperature (°C, °F)
This parameter indicates calculated odd gear clutch temperature based on engine coolant temperature.
Even Gear Clutch (K2) Temperature (°C, °F)
This parameter indicates calculated even gear clutch temperature based on engine coolant temperature.
Standard Mode (Drive Mode) (ON/OFF)
This parameter indicates driving mode switch signal from BCM through CAN.
ON: Driving mode switch is AUTO position.
OFF: Driving mode switch is not AUTO position.
Sport Mode (SPORT Mode) (ON/OFF)
This parameter indicates driving mode switch signal from BCM through CAN.
ON: Driving mode switch is SPORT position.
OFF: Driving mode switch is not SPORT position.
All Weather Mode (SNOW Mode) (ON/OFF)
This parameter indicates driving mode switch signal from BCM through CAN.
ON: Driving mode switch is SNOW position.
OFF: Driving mode switch is not SNOW position.
Auto/Manual Mode
This parameter indicates the gear shifting mode.
Automatic: Automatic gear shifting
Manual: Manual gear shifting
Odd Gears Clutch (K1) Over Temperature Time: Hour (hour)
This parameter indicates the states of odd gear clutch over temperature time.
Odd Gears Clutch (K1) Over Temperature Time: Minute (min)
This parameter indicates the states of odd gear clutch over temperature time.
Odd Gears Clutch (K1) Over Temperature Time: Second (sec)
This parameter indicates the states of odd gear clutch over temperature time.
Even Gears Clutch (K2) Over Temperature Time: Hour (hour)
This parameter indicates the states of even gear clutch over temperature time.
Even Gears Clutch (K2) Over Temperature Time: Minute (min)
This parameter indicates the states of even gear clutch over temperature time.
Even Gears Clutch (K2) Over Temperature Time: Second (sec)
This parameter indicates the states of even gear clutch over temperature time.
Odd Gears Clutch (K1) Kiss Point (Contact-point) Selftune

Executed: Contact-point learning is completed.
Not executed: Contact-point learning is not completed.
Even Gears Clutch (K2) Kiss Point (Contact-point) Selftune
This parameter indicates the states of even gear clutch contact-point learning.
Executed: Contact-point learning is completed.
Not executed: Contact-point learning is not completed.
Selection Positions Selfcalibrate Selftune
This parameter indicates the states of selfcalibrate learning.
Selftuned: Selfcalibrate learning is completed.
Not selftuned: Selfcalibrate learning is not completed.
Clutch Electrovalve Stabilized Current Selftune
This parameter indicates the states of clutch solenoid valve stabilized current learning.
Calculated: Solenoid valve stabilized current learning is completed.
Not calculated: Solenoid valve stabilized current learning is not completed.
Odd Gear Clutch (K1) Position Closed Selftune
This parameter indicates the states of odd gear clutch close learning.
Calculated: Clutch close learning is completed.
Not calculated: Clutch close learning is not completed.
Even Gear Clutch (K2) Closed Pressure Selftune
This parameter indicates the states of even gear clutch closed pressure learning.
Calculated: Clutch closed pressure learning is completed.
Not calculated: Clutch closed pressure learning is not completed.
Grid Thresholds Storage Selftune
This parameter indicates the states of grid thresholds storage learning.
Executed: Grid thresholds storage is completed.
Not executed: Grid thresholds storage learning is not completed.
Atmospheric Pressure (Barometric Pressure (kPa))
This parameter indicates barometric pressure signal from ECM through CAN.
Smart Driver Unit PWM Command (Electric Motor Command (%))
This parameter indicates control current value that smart drive unit outputs to electric motor.
Park Switch Status (ON/OFF)
This parameter indicates parking position detected by parking position sensor.
Engaged: Select lever in P range
Not-Engaged: Select lever in other range
