Gear Shift Control Lever and Cable Removal and Installation
Reference: Gear Shift Control Lever and Cable Components:K14C-6M/T Model
Removal
1)Remove floor console front cover.  
2)Remove gear shift lever cover.
3)For 4WD model, remove propeller shaft.  
4)Disconnect end (1) of gear shift control cable from pivot (2) of gear shift control lever assembly (3) while pushing release button (4) in arrow direction.
5)Disconnect end (5) of gear select control cable from pivot (6) of gear shift control lever assembly.
6)Disconnect gear shift and gear select control cables (7) from gear shift control lever assembly while pulling up quick joint (8) in arrow direction.
7)Remove gear shift control lever assembly mounting nuts (9) and remove gear shift control lever assembly from floor panel.
8)Disconnect gear shift and gear select control cables from transaxle while puling up quick joint (1) in arrow direction.
9)Remove cable grommet bolts (2) and cable clamp nuts (3), and then remove gear shift and select control cables from floor panel.
Installation
Reverse removal procedure noting the following points.
•Tighten each bolt and nut to specified torque.  
•Adjust gear select control cable.
