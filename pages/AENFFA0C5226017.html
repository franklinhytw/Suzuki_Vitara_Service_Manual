5B
Transaxle Right Case Disassembly and Reassembly
Reference: Manual Transaxle Unit Dismounting and Remounting:K14C-6M/T Model
Reference: Manual Transaxle Assembly Components:K14C-6M/T Model
Disassembly
1)Remove differential side oil seal from transaxle right case, if necessary.  
2)Remove differential side bearing outer race (1) using brass bar (2), if necessary.
3)Remove input shaft oil seal using special tool, if necessary.
Special Tool
(A): 09913–75830
4)Break oil channel (1) with flat-bladed screwdriver or the like and remove it.
5)Remove countershaft right bearing outer race using special tools, if necessary.
Special Tool
(A): 09942–15511
(B): 09944–96013
(C): 09921–26010
Reassembly
1)Install new input shaft oil seal (1) facing its spring side upward, if removed. Use special tool and hammer for installation and apply grease to oil seal lip.
“A”: Grease 99000–25011 (SUZUKI Super Grease A)
Special Tool
(A): 09913–75830
Distance from case surface to input shaft oil seal
“a”: 1.0 – 2.0 mm (0.04 – 0.07 in.)
2)Install countershaft right bearing outer race and new countershaft oil channel using special tool, if removed.
Special Tool
(A): 09913–85210
3)Install differential side bearing outer race using special tools and hammer, if removed.
Special Tool
(A): 09924–74510
(B): 09925–14520
4)Install new differential side oil seal to right case, if removed.
