1C
Registration for ECM Replacement
NOTE:
•For details of SUZUKI scan tool, refer to its Operator’s Manual.
•If the vehicle information cannot be downloaded to the new ECM from the replaced ECM using SUZUKI scan tool, perform the following items:
—Manual input of vehicle information
—Vehicle variant data registration
—DPF® after-sales regeneration.  
—Engine oil replacement and engine oil parameter reset  
—Injector calibration code
—EGR-1 data initialization
—EGR-2 data initialization
—A/F sensor dew point initialization
—DPF® driving pattern parameter initialization
—Rail pressure sensor data initialization
—Turbocharger test
—Immobilizer ID code registration
Vehicle Information Upload
1)When ignition is “OFF”, connect SUZUKI scan tool to DLC.
2)Set ignition “ON”.
3)Select “ECM replacement” in “Utility” menu.
4)Select “Vehicle information copy”.
5)Select “Upload vehicle information from ECM” and following instructions displayed on SUZUKI scan tool.
Vehicle Information Download
1)When ignition is “OFF”, connect SUZUKI scan tool to DLC.
2)Set ignition “ON”.
3)Select “ECM replacement” in “Utility” menu.
4)Select “Vehicle information copy”.
5)Select “Download vehicle information to ECM” and following instructions displayed on SUZUKI scan tool.
6)When “Download completed successfully” is displayed on SUZUKI scan tool, set ignition “OFF”, and wait for 30 sec. or more.
7)Register immobilizer ID code.
•Ignition switch model:  
•Keyless push start model:  
8)Set ignition “OFF” and leave for more 30 sec.
Vehicle Information Manual Input
NOTE:
•Use the “Vehicle Information Manual Input” procedure only when the “Vehicle Information Upload” and “Vehicle Information Download” procedures cannot be completed normally.
•Perform DPF® after-sales regeneration. Otherwise, the new ECM will not recognize the DPF® status, which will affect an original engine performance.
1)When ignition is “OFF”, connect SUZUKI scan tool to DLC.
2)Set ignition “ON”.
3)Select “ECM replacement” in “Utility” menu.
4)Select “Vehicle information manual input” and “Input vehicle variant data” manually and following instructions displayed on SUZUKI scan tool.
5)Register immobilizer ID code.
•Ignition switch model:  
•Keyless push start model:  
6)Select “Injector calibration code” manually and following instruction displayed on SUZUKI scan tool.
7)When “Inputting vehicle information was completed successfully” is displayed on SUZUKI scan tool, set ignition “OFF”, and wait for 30 seconds or more.
8)Set ignition “OFF” and leave for more 30 sec.
9)Perform DPF® after-sales regeneration.  
10)Replace engine oil and reset engine oil parameter.  
11)Perform the following items.
•EGR-1 data initialization  
•EGR-2 data initialization  
•A/F sensor dew point initialization  
•DPF® driving pattern parameter initialization  
•Turbocharger test  
12)Clear DTC.
