CMP Sensor Removal and Installation
Removal
1)Disconnect negative (–) cable at battery.
2)Remove engine cover.  
3)Disconnect connector from CMP sensor as follows:
a)Move connector lock pin (1) in arrow direction as shown in figure.
b)Push connector lock lever (2) and disconnect connector from CMP sensor (3).
4)Remove CMP sensor from camshaft housing.
Installation
Reference: CMP Sensor Inspection:D16AA
Reverse removal procedure noting the following points.
•Check O-ring for deterioration and damage. If check result is not OK, replace CMP sensor (1).
•Tighten CMP sensor bolt to specified torque.
Tightening torque
CMP sensor bolt (a): 9.0 N·m (0.92 kgf-m, 7.0 lbf-ft)
