Description of Speed Limiter System Components and Functions
Component Function
ECM and electric throttle body (M16A model / K14C model) or fuel injector (D16AA model) ECM performs speed limiter control based on the signal from cruise control switch.
ECM controls electric throttle valve opening (M16A model / K14C model) or fuel injection (D16AA model) not to exceed the maximum speed is set by the driver.
APP sensor Detects depressing degree of the accelerator pedal.
LIMIT switch This switch has a momentary contact type button to turn speed limiter function ON and OFF.
SET/– switch
•Current vehicle speed is inputted as the maximum speed when this switch is pressed (ON). If the vehicle speed is 30 km/h (20 mile/h) or less, the maximum speed is set at 30 km/h (20 mile/h), and if the vehicle speed is higher than 210 km/h (130 mile/h), the maximum speed is set at 210 km/h (130 mile/h) and the vehicle speed is decelerated to the maximum speed.
•When this switch is pressed (ON) while the speed limiter system is functioning, the maximum speed goes down. The maximum speed can be changed when the vehicle speed is at 0 km/h (0 mile/h).
RES/+ switch
•When this switch is pressed (ON) after cancellation of the speed limiter control, the system resumes functioning with the speed before the cancellation.
•When this switch is pressed (ON) while the speed limiter system is functioning, the maximum speed goes up. The maximum speed can be changed when the vehicle speed is at 0 km/h (0 mile/h).
CANCEL switch When this switch is pressed (ON), the speed limiter control is canceled.
“LIMIT” indicator In the state with ignition “ON” and speed limiter system in OFF, pressing LIMIT switch once and releasing it will activate the speed limiter system and ECM will cause indicator to light up.
“SET” indicator Lights up when the speed limiter system is functioning.
Information display
•Maximum speed is displayed while the speed limiter system is functioning.
•Maximum speed display blinks when the vehicle speed exceeds the maximum speed.
Warning buzzer (in combination meter) Warning buzzer sounds when the vehicle speed exceeds the preset maximum speed.
