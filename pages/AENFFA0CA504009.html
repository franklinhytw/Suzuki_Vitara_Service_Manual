Scan Tool Utility Table
SUZUKI scan tool has setting change function of keyless start control module as follows. For details of SUZUKI scan tool, refer to its Operator's Manual.
NOTE:
Some data in the table below may either be applicable or not applicable depending on the model.
Item Setting Description
Keyless Start System ON / OFF Setting of each inside and outside antenna (radio wave transmission allowed / not allowed) can be changed.
Answer-back (Outside buzzer) ON / OFF Setting (active / disabled) for outside warning buzzer during answer-back can be changed.
Remote Controller Area Warning ON / OFF Setting (active / disabled) for warning buzzer (in combination meter) during remote controller out of detection area alarm can be changed.
Remote Controller Battery Warning ON / OFF Setting (active / disabled) for remote controller battery run down warning can be changed.
Security Alarm ON / OFF Setting (active / disabled) for security alarm mode can be changed.
