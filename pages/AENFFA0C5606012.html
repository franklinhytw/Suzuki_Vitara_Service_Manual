Disassembly
1)Remove select lever assembly.  
2)Remove select lever knob screw (1).
3)Remove select lever knob (1), select lever indicator (2) and select lever boot (3).
4)Pushing locks (1) and remove select lever boot (2) from select lever knob (3).
5)For LHD model, remove shift lock manual release cap from select lever indicator.
6)Remove select lever illumination light assembly (1).
Reassembly
1)Check that shift lock is released and select lever can be shifted from P range to other range when shift lock manual release button is pushed.
2)Install select lever illumination light assembly (1) and fix wiring harness.
