ENG A-STOP Control Module
NOTE:
If communication between SUZUKI scan tool and ENG A-STOP control module is not possible, perform the following checks:
•Check communication condition between PC and VCI using SUZUKI scan tool function.
•Check DLC power circuit and ground circuit.
•Check DLC cable. If necessary, check DLC cable by substituting a known-good DLC cable.
•Check CAN communication system. (“Control Module Communication Bus Off” is detected in any control module.)

Scan Tool Data Definitions
Battery Temp (Battery Temperature, °C, °F)
This parameter indicates battery side and ambient temperature based on battery temperature sensor.
Battery Voltage (V)
This parameter indicates battery voltage inputted from “A-STOP” fuse to ENG A-STOP control module.
Battery Current (A)
This parameter indicates electric load value (current consumption) detected by electric load current sensor.
Brake Booster (Brake Booster Pressure, kPa)
This parameter indicates brake booster negative pressure.
Master Cyl Press (Master Cylinder Pressure, kPa)
This parameter indicates brake master cylinder pressure detected by master cylinder pressure sensor in ESP® control module.
Estimated Battery Temp (Estimated Battery Temperature, °C, °F)
This parameter indicates battery inside temperature estimated by ENG A-STOP control module.
Starter Drive Counter (Times)
This parameter indicates starting motor operation times. 5 times of engine starting are counted as one.
ECT (°C, °F)
This parameter indicates engine coolant temperature sent from ECM through CAN.
Vehicle Speed Sensor (Vehicle Speed, km/h, MPH)
The vehicle speed is calculated based on pulse signals from wheel speed sensors.
Accel Position (Accelerator Pedal Position, %)
This parameter indicates accelerator pedal position sent from ECM through CAN.
Engine Speed (rpm)
This parameter indicates engine speed sent from ECM through CAN.
Neutral Switch (Neutral Position Switch, ON/OFF)
This parameter indicates the state of neutral position switch.
Slope Range (%)
This parameter indicates slope angle (rate) calculated by ENG A-STOP control module.
DC-DC Signal 2
This parameter indicates the state of DC/DC converter.
Hood Switch1 (Hood Latch Switch 1, ON/OFF)
This parameter indicates the state of hood latch switch 1.
Hood Switch2 (Hood Latch Switch 2, ON/OFF)
This parameter indicates the state of hood latch switch 2.
Ignition Switch (ON/OFF)
This parameter indicates the state of ignition “ON”.
Starting Switch (ON/OFF)
This parameter indicates the state of ignition “START”.
For keyless push start model, this parameter indicates “OFF” at any time.
Half Clutch Switch (CPP No.2 Switch, ON/OFF)
This parameter indicates the state of CPP No.2 switch.
Clutch Switch (CPP No.1 Switch, ON/OFF)
This parameter indicates the state of CPP No.1 switch.
Idling Stop Prohibit SW (ENG A-STOP OFF Switch, ON/OFF)
This parameter indicates pushed state of ENG A-STOP OFF switch.
Charge Lamp SW (ON/OFF)
This parameter indicates the state of electricity generation by generator.
ON: Engine is stationary.
OFF: Engine is running or stationary by engine auto stop control (other than engine auto start inhibit control).
Brake SW (Brake Switch, ON / OFF)
This parameter indicates the state of brake light switch.
Starting Relay (+) (ST2 Relay (+), ON/OFF)
This parameter indicates the state of ST2 relay (+) for starting motor operation.
Starting Relay (–) (ST2 Relay (–), ON/OFF)
This parameter indicates the state of ST2 relay (–) for starting motor operation.
DC-DC Power2
This parameter indicates the state of DC/DC converter output at engine restart (cranking).
Starting Motor Drive (ON/OFF)
This parameter indicates the state of “Starting Motor” operation in “Active Test” mode on SUZUKI scan tool.
Stop Mode (ON/OFF)
This parameter indicates the state of the engine stop.
Starting Mode (ON/OFF)
This parameter indicates the state of the engine cranking.
Auto Stop Mode (ON/OFF)
This parameter indicates the state of the engine auto stop.
Normal Mode (ON/OFF)
This parameter indicates the state of the engine running.
Charging (ON/OFF)
This parameter indicates the state of the battery charge judged by ENG A-STOP control module.
Another System Malfunction (ON/OFF)
This parameter indicates the state of malfunction related to control modules other than ENG A-STOP control module.
Inhibition Condition (ON/OFF)
This parameter indicates the state of engine auto stop inhibition on ENG A-STOP control module.
ECM Status for A-STOP (Start/Inhibit/No)
This parameter indicates the state of ECM request for ENG A-STOP system.
Stop A-STOP Permission (TCM) (ON / OFF)
This parameter indicates the state of permission by TCM for engine auto stop control while vehicle is stopped.
Shift Position
This parameter indicates the state of range switch in transmission range sensor.
Driver Door Switch (ON/OFF)
This parameter indicates the state of driver door switch sent from BCM through CAN.
Driver Seatbelt Switch (Fasten/Unfasten/Unknown)
This parameter indicates the state of driver seat belt switch sent from BCM through CAN.
ABS/ESP® Status for A-STOP (Failure/Inhibit/Backup/Normal)
This parameter indicates the state of control on ESP® control module.
Reverse Switch (Back Up Light Switch, ON/OFF)
This parameter indicates the state of gear position “Reverse” sent from ESP® control module through CAN.
HVAC Status for A-STOP (ON/OFF)
This parameter indicates the state of engine start request from HVAC control module.
Engine Stop Request (OK/NG)
This parameter indicates the state of engine stop request from ENG A-STOP control module to ECM.
Battery Condition (OK/NG)
This parameter indicates the state of battery charge for execution of engine auto stop control.
Battery Temp Condition (Battery Temperature Condition, OK/NG)
This parameter indicates the state of estimated battery temperature for execution of engine auto stop control.
Brake Booster Condition (Brake Booster Pressure Condition, OK/NG)
This parameter indicates the state of brake booster negative pressure for execution of engine auto stop control.
Cylinder Press Condition (Cylinder Pressure Condition, OK / NG)
This parameter indicates the state of brake master cylinder pressure for execution of engine auto stop control.
Brake SW Condition (Brake Switch Condition, OK / NG)
This parameter indicates the state of brake light switch for execution of engine auto stop control.
Neutral SW Condition (Neutral Position Switch Condition, OK/NG)
This parameter indicates the state of gear position “Neutral” for execution of engine auto stop control.
Clutch SW Condition (Clutch Pedal Position Switch Condition, OK/NG)
This parameter indicates the state of clutch pedal position switch for execution of engine auto stop control.
Hood SW Condition (Hood Latch Switch Condition, OK/NG)
This parameter indicates the state of hood latch switch for execution of engine auto stop control.
Accel Pedal Condition (Accelerator Pedal Position Condition, OK/NG)
This parameter indicates the state of accelerator pedal position for execution of engine auto stop control.
Tilt Angle Condition (OK / NG)
This parameter indicates the state of vehicle inclination for execution of engine auto stop control.
Speed History Condition (OK/NG)
This parameter indicates the state of engine auto stop permission based on vehicle running speed after the previous engine auto stop.
TCM Demand Condition (OK / NG)
This parameter indicates the state of engine auto stop control permission sent from TCM.
ECM Demand Condition (OK/NG)
This parameter indicates the state of engine auto stop control permission sent from ECM.
ABS/ESP® Demand Condition (OK/NG)
This parameter indicates the state of engine auto stop control permission sent from ESP® control module.
HVAC Demand Condition (OK/NG)
This parameter indicates the state of engine auto stop control permission sent from HVAC control module.
Internal DC/DC Converter Voltage (V)
This parameter indicates the state of internal DC/DC converter output voltage.
