5A
TCM Learning Value Clearance
NOTICE:
If TCM learning value initialization is not performed after one of the following operations has been performed, there is a possibility that the vehicle will not be able to move.
Perform TCM learning value initialization whenever one of the following operations has been performed:
•TCM is removed or replaced.
•TCM software is reprogrammed.
•Valve body assembly is removed or replaced.
•A/T assembly is disassembled or replaced.
1)With ignition “OFF”, connect SUZUKI scan tool to DLC.
Special Tool
(A): SUZUKI scan tool (Suzuki SDT-II)
2)Set ignition “ON”.
3)Select “A/T Learned Initialize” in “Active test” menu, and follow instructions displayed on SUZUKI scan tool.
4)Set ignition “OFF”.
