Gear Shift and Select Shaft Assembly Disassembly and Reassembly
Reference: Gear Shift and Select Shaft Assembly Removal and Installation:D16AA-6M/T Model
Reference: Gear Shift and Select Shaft Assembly Components:D16AA-6M/T Model
Disassemble and reassemble gear shift and select shaft assembly referring to the following points and figure.
•Push spring pins out using special tools as shown below.
Special Tool
(A): 09925–78210
(B): 09922–89810
•Check component parts for wear, distortion or damage. If any defect is found, replace defective part with new one.
NOTICE:
The gear shift and select shaft may be bent if the spring pins are driven without using any support on the back of the spring pin fitting location.
Support the spring pin fitting location from behind with a wooden block when installing the spring pins.
