CKP Sensor Removal and Installation
Removal
1)Disconnect negative (–) cable at battery.
2)For 4WD model, remove starting motor (3).  
3)Disconnect connector (2) from CKP sensor (1).
4)Remove CKP sensor from cylinder block.
Installation
Reference: CKP Sensor Inspection:D16AA
Reverse removal procedure noting the following points.
•Clean CKP sensor and sensor rotor teeth before installation.
•Tighten CKP sensor bolt to specified torque.
Tightening torque
CKP sensor bolt (a): 9.0 N·m (0.92 kgf-m, 7.0 lbf-ft)
•Connect connector to CKP sensor securely.
