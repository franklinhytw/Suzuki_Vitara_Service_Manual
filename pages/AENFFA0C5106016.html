5A
Select Cable Removal and Installation
Reference: Select Cable Components
CAUTION:
Touching a hot exhaust system component will cause a burn.
Do not perform any service work on the select cable until the exhaust system is cool.
Removal
1)Disconnect select cable from select lever assembly.  
2)Remove battery, battery tray and battery bracket.
•M16A model:  
•K14C model:  
3)Disconnect eye end (1) from manual select lever pin, and then disconnect lock ring (2) from select cable bracket (3).
4)For 4WD model, remove propeller shaft.  
5)Remove select cable from dash panel.
Installation
Reverse removal procedure noting the following points.
•Apply grease to pins.  
•Tighten bolts and nuts to specified torque.  
•Adjust select cable.
