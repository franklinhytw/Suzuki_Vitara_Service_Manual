DTC P1321 / P1322
DTC Detecting Condition and Trouble Area
DTC detecting condition Trouble area
P1321: ENG A-STOP Battery Current Sensor “A” Circuit Low
Measured voltage of electric load current sensor circuit is lower than 0.2 V for specified time continuously.
•Electric load current sensor & battery temperature sensor circuit / connector
•Electric load current sensor & battery temperature sensor
•ENG A-STOP control module
P1322: ENG A-STOP Battery Current Sensor “A” Circuit High
Measured voltage of electric load current sensor circuit is higher than 4.8 V for specified time continuously.
Circuit Diagram
[A]: ENG A-STOP control module connector (View: [a]) A3: Battery temperature sensor signal circuit 2. ENG A-STOP control module
A1: Electric load current sensor power supply circuit A4: Electric load current sensor & battery temperature sensor ground circuit 3. Electric load current sensor
A2: Electric load current sensor signal circuit 1. Electric load current sensor & battery temperature sensor 4. Battery temperature sensor
DTC Confirmation Procedure
1)Set ignition “ON”.

2)Disconnect electric load current sensor & battery temperature sensor connector.
3)Check for proper terminal connection to electric load current sensor & battery temperature sensor connector.
4)If connections are OK, set ignition “ON”.
5)Check that voltage between “A1” and “A4” is approx. 5 V.

Go to Step 4.

Is check result OK?
Repair “A4” circuit. If DTC P1321 or P1322 is still detected, replace ENG A-STOP control module and recheck for DTC.

Electric load current sensor power supply circuit check
1)Set ignition “OFF”.
2)Disconnect “E24” connector.
3)Check for proper terminal connection to “E24” connector.
4)If connections are OK, check the following points.
•Resistance of “A1” circuit: less than 1 Ω
•Resistance between “A1” circuit and ground: infinity
•Resistance between “A1” circuit terminal and other terminal at electric load current sensor & battery temperature sensor connector: infinity
•Voltage of “A1” circuit: approx. 0 V (When ignition is “ON”)
Is check result OK?
Replace ENG A-STOP control module and recheck for DTC.  
Repair or replace defective wire harness.

•Resistance between “A2” circuit terminal and other terminal at electric load current sensor & battery temperature sensor connector: infinity
•Voltage of “A2” circuit: approx. 0 V (When ignition is “ON”)
