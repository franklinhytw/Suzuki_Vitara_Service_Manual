MAP Sensor Removal and Installation
Removal
1)Disconnect negative (–) cable at battery.
2)Disconnect connector from MAP sensor.
3)Remove MAP sensor (1) from intake manifold (2).
Installation
Reference: MAP Sensor Inspection:K14C
Reverse removal procedure noting the following points.
•Check that O-ring is free from damage.
•Tighten MAP sensor bolt (1) to specified torque.
Tightening torque
MAP sensor bolt (a): 11 N·m (1.1 kgf-m, 8.5 lbf-ft)
