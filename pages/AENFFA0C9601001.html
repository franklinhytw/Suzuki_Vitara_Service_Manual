Power Door Lock Description
•Power door lock system for ignition switch model is controlled by keyless entry control module and BCM, and the system for keyless push start model is controlled by keyless start control module and BCM.
•Power door lock system consists of keyless entry / keyless start control module, BCM, door lock actuators and power window main switch.
•Door lock actuators are integrated into door lock assembly. Front door lock assembly also integrates the door lock switch.
•Door lock actuators are all covered for anti-theft purpose.
