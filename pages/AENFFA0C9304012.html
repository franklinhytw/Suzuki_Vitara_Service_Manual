B1851: Front Parking Sensor Communication Bus Error
Combination meter cannot send / receive communication data to / from front parking sensor.
•Parking sensor system control module
•Parking sensor
•Communication circuit
B1852: Rear Parking Sensor Communication Bus Error
Combination meter cannot send / receive communication data to / from rear parking sensor.

DTC Troubleshooting
NOTE:
The following procedure describes how to check DTC B1852. Follow the same procedure for DTC B1851.
Step Action Yes No
1
Circuit check
1)Check that ignition is “OFF”.
2)Disconnect parking sensor system control module connectors and all parking sensor connectors.
3)Check for proper connection parking sensor system control module connector and all parking sensor connectors.
4)If connections are OK, check circuits between parking sensor system control module connector and all parking sensor connectors for open circuit, short to ground or short to power supply.
Is check result OK?
Go to Step 2.
Repair defective wire harness.
2
Parking sensor (rear-left) check
1)Connect parking sensor system control module connector and all parking sensor connectors.
2)Replace parking sensor (rear-left).  
3)Set ignition “ON”.
4)Check DTC.
Is DTC B1852 still detected?
Go to Step 3.
End.
3
Parking sensor (rear-center-left) check
1)Replace parking sensor (rear-center-left).  
2)Set ignition “ON”.
3)Check DTC.
Is DTC B1852 still detected?
Go to Step 4.
End.
4
Parking sensor (rear-center-right) check
1)Replace parking sensor (rear-center-right).  
2)Set ignition “ON”.
3)Check DTC.
Is DTC B1852 still detected?
Go to Step 5.
End.
5
Parking sensor (rear-right) check
1)Replace parking sensor (rear-right).  
2)Set ignition “ON”.
3)Check DTC.
Is DTC B1852 still detected?
Replace parking sensor system control module.  
End.
