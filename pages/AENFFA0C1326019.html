CMP Sensor Removal and Installation
Reference: CMP Sensor Inspection:K14C
NOTICE:
If CMP sensor is removed once, do not reuse it.
Always install a new one.
Removal
1)Disconnect negative (–) cable at battery.
2)Remove engine cover.  
3)Remove air cleaner suction hose, if necessary.  
4)Disconnect connector from CMP sensor.
5)Remove CMP sensor (1) from thermostat case.
Installation
Reverse removal procedure noting the following points.
•Apply engine oil to O-ring of CMP sensor.
•Tighten CMP sensor bolt (1) to specified torque.
Tightening torque
CMP sensor bolt (a): 11 N·m (1.1 kgf-m, 8.5 lbf-ft)
