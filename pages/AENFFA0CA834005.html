10H
Communication Check
Keyless Entry / Keyless Start Control Module
1)Set ignition “OFF”.
2)Connect “G16” connector to keyless entry / keyless start control module.
3)Set ignition “ON”.
4)Select “Communication Bus Check” menu of SUZUKI scan tool.
5)Check the CAN communication that control modules, sensors and keyless entry / keyless start control module which have been already connected are in normal condition.  
6)If any malfunction is found, replace keyless entry / keyless start control module and recheck DTC.
