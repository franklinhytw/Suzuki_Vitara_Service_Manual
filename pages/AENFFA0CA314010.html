P1644: Immobilizer ID Mismatched
Verification error between ECM and keyless entry control module.
•Ignition key
•ECM ID code is not registered in keyless entry control module
•Ignition key is not registered
•Immobilizer control module circuit / connector
•Immobilizer control module
•Keyless entry control module power supply / ground circuit
•ECM
•Keyless entry control module
DTC Troubleshooting
NOTE:
If keyless entry control module is used in another vehicle before, ignition key cannot be registered. Always install new keyless entry control module whenever replacement is necessary.
