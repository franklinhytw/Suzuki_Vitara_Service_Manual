Lighting System Description
Auto-On Headlight System (If Equipped)
•The auto-on headlight system is a system to control the automatic ON / OFF of headlight, clearance lights, tail lights, license plate lights and adjust interior illumination optimally according to ambient brightness when lighting switch is in “AUTO” position.
•BCM controls the operation of the auto-on headlight system based on light intensity signals from rain / light sensor. For general description of rain / light sensor, refer to Rain / Light Sensor Description (If Equipped).
•With ignition “ON” and the lighting switch in “AUTO”, BCM controls operation of the headlights, clearance lights, tail lights and license plate lights in the following patterns according to the outside light condition.
Lighting switch position Ignition Outside light condition Headlights Clearance lights, tail lights, license plate lights
OFF Any Any condition OFF OFF
Clearance light Any Any condition OFF ON
Headlight Any Any condition ON ON
AUTO “ON” Clear OFF OFF
AUTO “ON” Temporarily dark under clear condition ON ON
AUTO “ON” Dim ON ON
AUTO “ON” Dark ON ON
AUTO “ON” Temporarily clear under dark condition ON ON
AUTO “OFF” Any condition OFF OFF
•The auto-on / auto-off intensity of outside light is programmable for the following three settings:
—Auto-on at standard outside light intensity
—Auto-on at lower outside light intensity than standard
—Auto-on at lowest outside light intensity in the three settings
•In addition to auto-on / auto-off operation responding to the outside light intensity, the auto-on headlight system can automatically turn on / off the headlights interlocked with the rain-sensing wiper system operation according to settings.
Auto-on headlight operation interlocked with rain-sensing wiper operation
Auto-on headlight operation interlocked with rain-sensing wiper operation is programmable for the following three settings:
•Auto-on under heavy rainfall condition
•Auto-on when the rain / light sensor senses light and heavy rainfall
•No auto-on under any rainfall condition
DRL System (If Equipped)
•The DRL system consists of BCM, DRL relay, lighting switch, generator and LED DRLs or DRL bulbs.
•While the engine is running, BCM executes the LED DRLs or DRL bulbs to ON as follows.
—Lighting switch is OFF
—Lighting switch is in clearance position and front fog light is OFF
