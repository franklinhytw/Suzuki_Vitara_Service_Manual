5A
Neutral Position Calibration
NOTICE:
If neutral position calibration is not performed after one of the following operations has been performed, there is a possibility that the vehicle will not be able to move.
Perform neutral position calibration whenever one of the following operations has been performed:
•TCM is removed or replaced.
•TCM software is reprogrammed.
•Valve body assembly is removed or replaced.
•A/T assembly is disassembled or replaced.
1)If TCM is replaced with new one, install software to TCM. Refer to Operator's Manual of SUZUKI scan tool for further details.
2)With ignition “OFF”, connect SUZUKI scan tool to DLC.
3)Set ignition “ON” and shift select lever to N range.
4)Select “Neutral position calibration” in “Utility” menu, and follow instructions displayed on SUZUKI scan tool.
5)When ignition is “OFF”, disconnect SUZUKI scan tool from DLC.
