—Check CAN communication system. (“Control Module Communication Bus Off” is detected in any control module.)

Scan Tool Data Definitions
Battery Voltage (V)
This parameter indicates battery voltage detected by RBS control module.
Detecting Distance (m)
This parameter indicates distance to a vehicle ahead when millimeter wave radar detects the vehicle.
RBS Warning Timing (FAR / NEAR)
This parameter indicates the state of FAR/NEAR setting switch.
Radar Brake Support Status (ON / OFF)
This parameter indicates whether or not fontal collision warning function, brake assist function and automatic brake function are active.
Object Detection Status (ON / OFF)
This parameter indicates whether or not millimeter wave radar detects a vehicle ahead.
Radar Visibility (OK / NG)
This parameter indicates state of millimeter wave radar visibility judged by RBS control module.
RBS Control Module Temperature Condition (OK / NG)
This parameter indicates internal temperature of RBS control module.
RBS Control Module Voltage Condition (OK / NG)
This parameter indicates power supply voltage of RBS control module.
ABS/ESP® Control Module Condition (OK / NG)
This parameter indicates state of automatic brake function control transmitted from ESP® control module.
Outside Air Temperature Condition (OK / NG)
This parameter indicates state of outside air temperature sensor transmitted from BCM.
RBS OFF Switch (ON / OFF)
This parameter indicates the state of RBS OFF switch.
