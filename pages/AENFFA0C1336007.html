1C
A/F Sensor Parameter Initialization
After replacing A/F sensor-1 with new one, perform the following procedure.
1)When ignition is “OFF”, connect SUZUKI scan tool to DLC.
2)Set ignition “ON”.
3)Execute “A/F sensor parameter initialization” in “Utility” menu following instructions displayed on SUZUKI scan tool.
4)When ignition is “OFF”, disconnect SUZUKI scan tool from DLC.
