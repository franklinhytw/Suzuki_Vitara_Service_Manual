Antenna Inspection
Using SUZUKI Scan Tool
1)Connect SUZUKI scan tool to DLC with ignition “OFF”.
2)Set ignition “ON”.
3)Select “Keyless Start Antenna” function in “Active Test” mode.
4)Check that each antenna operates.
If check result is not as specified, check applicable circuit for open or short.  
If check result is as specified, replace applicable antenna.
