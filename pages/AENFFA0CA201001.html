10B
BCM General Description
BCM incorporates relays and controllers to control the following parts/systems.
NOTE:
*: The number of retard flashing times can be set up for momentary switching on and off of turn signal switch in case of necessity such as lane change.
•Power door lock
•Rear end door opener
•Clearance light
•Auto-on headlight (LED headlight model)
•Front fog light (if equipped)
•Rear fog light (if equipped)
•DRL (if equipped)
•Turn signal light*
•Interior light
•Windshield wiper (LED headlight model)
•Rear wiper
•Rear defogger
•Seat belt reminder
•Warning buzzer (in combination meter)
•Shift lock (A/T model / twin clutch system model)
