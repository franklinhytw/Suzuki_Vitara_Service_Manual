Deadlock System Description (If Equipped)
The deadlock system prevents the door locks from being tampered with and unlocked when the system is activated using the key, transmitter, remote controller or driver door request switch.
Activation
•The deadlock system is activated and all doors are locked when the driver door lock is turned twice in the locking direction within 3 seconds using the key.
•The deadlock system is activated and all doors are locked when the remote controller’s lock button or the driver door request switch is pressed twice within 3 seconds.
Deactivation
•The deadlock system is deactivated only for the driver door and the driver door is unlocked when the driver door lock is turned once in the unlocking direction using the key. When the driver door lock is turned twice in the unlocking direction, the system is deactivated for all doors and the doors are unlocked.
•The deadlock system is deactivated only for the driver door and the driver door is unlocked when the remote controller’s lock button or the driver door request switch is pressed once. When the button or switch is pressed twice, the system is deactivated for all doors and the doors are unlocked.
