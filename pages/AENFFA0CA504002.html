10E
Keyless Start Control Module Self-Diagnosis Function
Description
Keyless start control module monitors conditions of the system components and its circuit. When an abnormal condition in the system occurs, keyless start control module performs control as follows.
Self-Diagnosis Function
Keyless start control module is equipped with self-diagnosis function that stores DTC for abnormalities detected in keyless start system and immobilizer control system. Stored DTC is saved until abnormal condition is corrected.
Warning Function
•When a malfunction of keyless start system and immobilizer control system is detected, keyless start control module turns on theft deterrent light and/or immobilizer indicator light. And, depending on the malfunction, a message is shown in information display to warn the driver.
NOTE:
When DTC B1194 is detected, immobilizer indicator light stays on until normal condition is detected at least twice consecutively.
•After detecting a malfunction, theft deterrent light and immobilizer indicator light are turned on. However they are turned off when normal condition is detected.

Fail-Safe Control
Keyless start control module performs fail-safe control when an abnormal condition related to keyless start system is detected and stops all controls to protect system. Details of fail-safe control vary depending on the detected DTC.  
NOTE:
When a malfunction of immobilizer control system is detected, keyless start control module does not perform fail-safe control.
