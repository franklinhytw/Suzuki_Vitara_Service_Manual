B1194: Starting Motor Relay Malfunction
Inconsistency between starting motor control relay monitor input voltage and starting motor control relay ON condition is continued for longer than specified time.
•Starting motor control relay circuit / connector
•Starting motor control relay
•“ST” fuse
•“ST SIG2” fuse
•CPP No.1 switch circuit / connector (M/T model)
•CPP No.1 switch (M/T model)
•Transmission range sensor circuit / connector (A/T model)
•TCM (including transmission range sensor) (A/T model)
•Keyless start control module

DTC Confirmation Procedure
1)Start engine.
2)Stop engine.
3)Repeat Step 1) – 2) once.
DTC Troubleshooting
NOTE:
This DTC might be detected when battery is run down or the voltage is very low.
Step Action Yes No
1
DTC check
Is DTC B1125 detected?
Go to troubleshooting for applicable DTC.
Go to Step 2.
2
Fuse check
1)Check “ST” and “ST SIG2” fuses.
Is check result OK?
Go to Step 3.
Replace “ST” and/or “ST SIG2” fuse.
3
Starting motor control relay check
1)Check starting motor control relay.  
Is check result OK?
Go to Step 4.
Replace starting motor control relay.
4
Starting motor control relay signal circuit check
1)Set ignition “OFF”.
2)Disconnect “E01”, “E484” and starting motor connectors.
3)Check for proper terminal connection to “E01”, “E484” and starting motor connectors.
4)If connections are OK, check the following points.
•Resistance of “A1” and “A2” circuits: less than 1 Ω
•Resistance between each of “A1” and “A2” circuits and ground: infinity
•Resistance between “A1” circuit terminal and each terminal at starting motor control relay connector: infinity
Is check result OK?
Replace keyless start control module and recheck DTC.  
Repair or replace defective wire harness.
