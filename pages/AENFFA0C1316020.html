Knock Sensor Removal and Installation
Reference: Knock Sensor On-Vehicle Inspection:M16A
Removal
1)Disconnect negative (–) cable at battery.
2)Remove intake manifold.  
3)Disconnect knock sensor connector.
4)Remove knock sensor (1) from cylinder block.
Installation
Reverse removal procedure noting the following points.
•Install knock sensor (1) as shown in figure.
•Tighten knock sensor bolt (2) to specified torque.
Tightening torque
Knock sensor bolt (a): 23 N·m (2.3 kgf-m, 17.0 lbf-ft)
“a”: 30°
