DTC Detecting Condition and Trouble Area
DTC detecting condition Trouble area
B1095-01: Side Rear Impact Sensor No Response (Driver Side)
Driver-side rear side-impact sensor has communicated with no response.
•Driver-side and/or passenger-side rear side-impact sensor circuit
•Driver-side and/or passenger-side rear side-impact sensor
•SDM
B1095-95: Side Rear Impact Sensor Communication Data Inconsistent (Driver Side)
Driver-side rear side-impact sensor has communicated with inconsistent ID code.
B1096-01: Side Rear Impact Sensor No Response (Passenger Side)
Passenger-side rear side-impact sensor has communicated with no response.
B1096-95: Side Rear Impact Sensor Communication Data Inconsistent (Passenger Side)
Passenger-side rear side-impact sensor has communicated with inconsistent ID code.

Rear side-impact sensor check
1)Check that ignition is “OFF”.

3)Set ignition “ON”.
4)Check for DTC.

Replace SDM and recheck DTC.  
End.
NOTE:
Upon completion of inspection and repair work, perform the following items.
•Reconnect all air bag system components and check that all components are properly mounted.
•Clear DTCs, referring to DTC Clearance if any DTC is detected.
•Repeat Air Bag Diagnostic System Check to check that the trouble has been solved.
