10E
Rear End Door Opener and Request Switch Removal and Installation
Removal
1)Disconnect negative (–) cable at battery.
2)Remove rear end door trim.  
3)Disconnect rear end door opener and request switch connector (1).
4)Remove rear end door license garnish (1).
5)Remove screws, and then remove rear end door opener and request switch (1).
Installation
Reference: Request Switch Inspection
Reverse removal procedure noting the following point.
•Tighten rear end door license garnish nuts to specified torque.
Tightening torque
Rear end door license garnish nut: 5.5 N·m (0.56 kgf-m, 4.0 lbf-ft)
