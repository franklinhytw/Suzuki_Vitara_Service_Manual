Self-Diagnosis Function
SDM also has self-diagnosis function that memorizes DTC when an error is detected. After understanding the following descriptions, perform troubleshooting referring to Air Bag Diagnostic System Check when an error is detected.
•SDM always monitors system while ignition is “ON”. If an error is detected, it notifies driver of error by turning on “AIR BAG” warning light (1) and detected error is stored in SDM memory as DTC.
•“AIR BAG” warning light turns on for about 6 seconds with ignition “ON” to indicate that it operates properly and turns off if no error is found in air bag system.
