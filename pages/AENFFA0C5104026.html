P0717: Input / Turbine Speed Sensor “A” Circuit No Signal
Input shaft speed sensor signal is not input even if output shaft speed sensor signal is input.
(1 D/C detection logic)
•Input shaft speed sensor
•Solenoid harness with ATF temperature sensor
•TCM
P1738: Input / Turbine Speed Sensor “A” Circuit Low
Input shaft speed sensor signal circuit is open or shorted to ground.
(1 D/C detection logic)
P1739: Input / Turbine Speed Sensor “A” Circuit High
Input shaft speed sensor signal circuit is shorted to power supply.
(1 D/C detection logic)

DTC Confirmation Procedure
P0717
WARNING:
If you do not select a suitable route and take proper precautions when performing a road test, an accident can occur.
•Select a level road with no traffic to minimize the risk of accident.
•Perform the test extremely carefully.
•Perform the road test using 2 persons, a driver and a tester.
NOTE:
Check that the following condition is met in this “DTC Confirmation Procedure”.
•DTC U0073 or U0100 is not detected.
1)Warm up engine to normal operating temperature.
2)Drive vehicle at approx. 30 km/h (19 mile/h) for 5 minutes.
P1738 / P1739
1)Set ignition “ON” for 10 sec.

3
Solenoid harness check
1)Remove TCM.  
2)Remove valve body assembly.  
3)Disconnect input shaft speed sensor connector.
4)Check for proper terminal connections to solenoid harness connector and input shaft speed sensor connector.
5)If connections are OK, check the following points.
•Resistance of “A1” and “A2” circuits: less than 1 Ω
•Resistance between each of “A1” and “A2” circuits and ground: infinity
•Resistance between “A1” circuit terminal and other terminal at solenoid harness connector: infinity
•Resistance between “A2” circuit terminal and other terminal at solenoid harness connector: infinity
•Voltage of “A1” and “A2” circuits: approx. 0 V (When ignition is “ON”)
Is check result OK?
Go to Step 4.
Replace solenoid harness with ATF temperature sensor.  
4
Input shaft speed sensor check
1)Check input shaft speed sensor.  
Is check result OK?
Substitute a known-good TCM and recheck DTC.  
Replace input shaft speed sensor.
