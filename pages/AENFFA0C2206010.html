Suspension Arm Front Bushing Removal and Installation
Reference: Suspension Arm Components
Reference: Suspension Arm Removal and Installation
NOTE:
Suspension arm rear bushing is not individually replaceable. Replace suspension arm if the bushing is defective.
Removal
1)Cut off flange of suspension arm front bushing (rubber) (1) using knife.
2)Push out suspension arm front bushing (1) using hydraulic press (2) and special tools.
Special Tool
(A): 09943–76311
(B): 09913–75821
Installation
1)Press-fit suspension arm front bushing (1) using special tools and hydraulic press (2).
NOTICE:
Reused bushing will not fit properly.
Always use a new bushing.
Special Tool
(A): 09943–76311
(B): 09913–75821
NOTE:
Before installing bushing, apply soap water on its circumference to facilitate installation.
2)Press-fit suspension arm front bushing (1) so that dimensions “a” and “b” in figure become equal.
