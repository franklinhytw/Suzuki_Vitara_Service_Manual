5A
TCM Learning Value Initialization
NOTICE:
If TCM learning value initialization is not performed after one of the following operations has been performed, there is a possibility that the vehicle will not be able to move.
Perform TCM learning value initialization whenever one of the following operations has been performed:
•TCM is removed or replaced.
•TCM software is reprogrammed.
•Valve body assembly is removed or replaced.
•A/T assembly is disassembled or replaced.
1)Drive vehicle so that ATF temperature reaches normal operating temperature (65 to 110 °C (149 to 230 °F)).
NOTE:
Engine coolant temperature rises when racing engine at P or N range. But ATF temperature does not rise at P or N range.
Drive vehicle to raise ATF temperature.
2)Perform garage shift learning as follows:
a)Stop vehicle with engine running and place it horizontally.
b)Keep engine idling, depress brake pedal fully and shift select lever to N range for 3 seconds.
c)Shift select lever to D range for 3 seconds.
d)Repeat Steps b) – c) 5 times.
e)Shift select lever to N range for 3 seconds, and then shift select lever to R range for 3 seconds.
f)Repeat Step e) 5 times.
3)Perform gear shift control learning as follows:
a)Drive vehicle at D range with throttle position 25 to 35% until vehicle speed exceeds 70 km/h (43 mile/h) at 5 speed.
b)Release accelerator pedal and, with engine brake applied, allow vehicle to coast for 60 sec. and stop vehicle.
c)Repeat Steps a) – b) 10 times.
