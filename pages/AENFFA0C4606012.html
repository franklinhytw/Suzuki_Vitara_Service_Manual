Front and Rear Wheel Speed Sensor Encoder Inspection
Reference: Front Wheel Speed Sensor Encoder Removal and Installation
Reference: Rear Wheel Speed Sensor Encoder Removal and Installation
NOTE:
Wheel speed sensor encoder is an integral part of wheel hub component. Replace front wheel bearing or rear wheel hub assembly if wheel speed sensor encoder is defective.
Front Wheel Speed Sensor Encoder
Check front wheel speed sensor encoders for crack, damage or deformation.
Replace front wheel bearing if faulty condition is found in encoder.
Rear Wheel Speed Sensor Encoder
Check rear wheel speed sensor encoder, referring to Front and Rear Wheel Speed Sensor On-Vehicle Inspection.
