Identification Plate
Identification plate (1) is installed at the center pillar lower side on the right or left side body panel.
The identification plate has the following information:
•VIN (2)
•Vehicle type (3)
•Engine type (4)
•Color code (5)
•Model code (6) including the model information shown in Market Code Description below.
[A]: Type A [B]: Type B
Market Code Description
The composition of the market code and the description of digit are as follows.
•The model code is composed of 9 characters.
•The 8th and 9th characters indicate the market code For example, “71” means market code “P71”.
The market code is used in this manual to facilitate exact identification of the vehicle specification.
