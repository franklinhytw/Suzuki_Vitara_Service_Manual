DTC detecting condition Trouble area
C1640: RBS Control Temporary Inhibition
RBS control module detects this DTC if any one of the following conditions is met.
•Poor visibility of millimeter wave radar.
•High temperature in RBS control module.
•Power supply for RBS control module is not between 10.0 and 15.5 V for 2 seconds continuously.
•RBS control module received automatic brake temporary stop signal from ESP® control module.
•RBS control module received invalid outside air temperature sensor signal from BCM.
•Millimeter wave radar surface or millimeter wave radar cover
•RBS control module power supply / ground circuit
•Charging system
•Outside air temperature sensor
•RBS control module

System Description
When RBS control module judges that the system impairs the original performance, RBS control module suspends frontal collision warning function, brake assist function, automatic brake function and adaptive cruise control, and outputs DTC C1640. However, when the cause of the suspension is eliminated, RBS control module turns off RBS indicator light, clears the DTC and resumes the system function automatically.
DTC Confirmation Procedure
1)Set ignition “ON”.
DTC Troubleshooting
NOTE:
•DTC C1640 can be detected as a current DTC but it is not stored as a history DTC.
•When any malfunction is detected on each parameter in the following table, “NG” indication is maintained. Therefore, after resuming the system to be normal, the factor of the malfunction can be checked in “Data List” of SUZUKI scan tool. The “NG” indication is maintained until the DTC is cleared.
1)Check each parameter in the table below using SUZUKI scan tool.
2)Diagnose the defective parameter according to diagnosis flow. If diagnosis flow does not correspond to the parameter, perform RBS System Check.

Diagnosis flow: [a]
NOTE:
•With the radiator grill installed, the millimeter wave radar cover cannot be removed. It is necessary to remove the radiator grill for visual inspection of the millimeter wave radar device (RBS control module).
•Even if millimeter wave radar surface is not soiled, RBS control module may detect poor visibility on millimeter wave radar and output DTC C1640 because diffused reflection is caused due to road conditions such as inside of tunnel, a road with concrete walls on both sides or weather conditions such as heavy rain or snow. If the vehicle is driven for about 2 minutes after the above conditions have improved, RBS control module clears the DTC and resumes the system function automatically.
