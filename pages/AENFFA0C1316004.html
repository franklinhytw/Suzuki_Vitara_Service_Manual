Electric Throttle Control System Calibration
NOTE:
•If the service described under Description of Electric Throttle Control System Calibration:M16A is performed, calibrate electric throttle control system as follows.
•Check that the following conditions are satisfied when performing this procedure.
—Ambient temperature > 5 °C (41 °F)
—DTC is not detected.
—Accelerator pedal position: Idle position
—5 °C (41 °F) < ECT < 100 °C (212 °F)
—Battery voltage > 10 V
1)Set ignition “ON” for 60 sec.
2)Set ignition “OFF”.
3)After completing this calibration, check that DTC is not detected in ECM.
If DTC is detected, go to troubleshooting for applicable DTC.
