Rear End Door Latch On-Vehicle Inspection
•Check that rear end door opens and closes smoothly.
•Check that rear end door closes completely.
Rear End Door Opener Actuator (If Equipped)
1)Remove rear end door trim.  
2)Disconnect rear end door lock connector (1) from rear end door lock assembly (2).
3)Check operation of actuator with battery voltage applied to the following terminals of rear end door lock assembly connector.
Replace rear end door lock assembly if defective.
Rear end door opener actuator operation
Rear End Door Switch
1)Remove rear end door trim.  
2)Disconnect rear end door lock connector (1) from rear end door lock assembly (2).
3)Check that there is continuity between the following terminals when rear end door is opened.
4)Check that there is no continuity between terminals when rear end door is closed.
If check result is not as specified, replace rear end door lock assembly.
