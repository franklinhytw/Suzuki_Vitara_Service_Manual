Rear Wheel Hub Assembly Removal and Installation
Reference: Rear Wheel Hub Components:Rear
Removal
1)Hoist vehicle and remove rear wheel.
2)For 4WD model, remove drive shaft nut.  
3)Remove rear brake disc.  
4)For 2WD model, remove rear wheel speed sensor from rear wheel hub assembly.  
For 4WD model, disconnect rear wheel speed sensor connector (1).
5)Remove rear wheel hub assembly bolts (1), and then remove rear wheel hub assembly and rear brake disc dust cover from rear axle.
NOTE:
For 4WD model, lightly tap rear drive shaft end of wheel hub side when it is difficult to remove rear wheel hub assembly.
[A]: 2WD model [B]: 4WD model
Installation
Reverse removal procedure noting the following points.
•For 2WD model, install rear wheel hub assembly, facing its hole for the wheel speed sensor up.
For 4WD model, install rear wheel hub assembly, positioning the wheel speed sensor to the vehicle front side.
•Tighten new rear wheel hub assembly bolts (1) to specified torque.
Tightening torque
Rear wheel hub assembly bolt (a): 88 N·m (9.0 kgf-m, 65.0 lbf-ft)
[A]: 2WD model [B]: 4WD model
•For 4WD model, tighten new drive shaft nut to specified torque.
