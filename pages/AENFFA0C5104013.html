5A
Stall Test
This test is to check overall performance of A/T and engine by measuring stall speed in D and R ranges. Perform this test only when ATF is at normal operating temperature and its level is adequate.
NOTICE:
If proper precautions are not taken when performing stall tests, a transaxle failure may result due to an excessive rise in ATF temperature.
•Do not continue running the engine for longer than 5 seconds at stall speed.
•After performing a stall test, allow the engine to run it at idle speed for longer than 1 minute before performing another stall test.
NOTE:
Repeat test 3 times and take average of those data.
1)Connect SUZUKI scan tool to DLC.
2)Select “Data list” mode on scan tool.
3)Drive vehicle so that ATF temperature reaches normal operating temperature (50 to 80 °C (122 to 176 °F)).  
NOTE:
Engine coolant temperature rises when revving engine in P or N range, but ATF temperature does not rise in P or N range.
Drive vehicle to raise ATF temperature.
4)Apply parking brake and block wheels.
5)Start engine with select lever in P range.
6)Depress brake pedal fully.
7)Shift select lever to D range and depress accelerator pedal fully while watching tachometer. Read engine rpm quickly when it has become constant (stall speed).
8)Release accelerator pedal immediately after stall speed is checked.
9)In the same way, check stall speed in R range.
10)Stall speed should be within the following specification.
Engine stall speed
M16A model: 2,300 – 2,600 rpm
K14C model: 2,230 – 2,530 rpm
