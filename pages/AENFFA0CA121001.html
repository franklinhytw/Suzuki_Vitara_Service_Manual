Adaptive Cruise Control System Construction
•In addition to a constant-speed control like a conventional cruise control system, the adaptive cruise control system has a constant headway gap control which keeps distance to a vehicle ahead.
•Adaptive cruise control system consists of RBS control module, cruise control switch, ECM, ESP® control module, steering angle sensor, combination meter, ENG A-STOP control module, TCM (A/T model / twin clutch system model) and BCM.
•The adaptive cruise control system adopts a millimeter (radio) wave radar system as a sensor to detect distance to a vehicle ahead and relative speed.
•Distance switch is incorporated in the cruise control switch and located in the steering wheel right side. Using this switch, vehicle distance can be set to any one of “Long”, “Middle” and “Short” modes.
•For driver’s visual recognition, the state of the control by the adaptive cruise control system is shown on the information display in the combination meter.
