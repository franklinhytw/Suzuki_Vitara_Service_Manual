9F
Rear End Door Lock Removal and Installation
Reference: Rear End Door Lock Components
Reference: Rear End Door Latch On-Vehicle Inspection
Removal
1)Remove rear end door trim.
2)Disconnect rear end door lock connector (1) from rear end door lock assembly (2).
3)Remove rear end door lock bolts (3) and remove rear end door lock assembly.
Installation
Reverse removal procedure noting the following points.
•Tighten rear end door lock bolts to specified torque in numerical order (“1” – “3”).
Tightening torque
Rear end door lock bolt (a): 10 N·m (1.0 kgf-m, 7.5 lbf-ft)
•Adjust rear end door latch striker as follows if necessary.
a.Remove tail end trim cap (1).
b.Loosen rear end door latch striker screws (2).
c.Adjust rear end door latch striker (3) position so that rear end door can be opened and closed smoothly.
d.Tighten rear end door latch striker screws to specified torque.
Tightening torque
Rear end door latch striker screw (a): 28 N·m (2.9 kgf-m, 21.0 lbf-ft)
